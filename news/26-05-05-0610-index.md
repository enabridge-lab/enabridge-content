# Daily Brief — 26-05-05 รอบ 06:10 (Bangkok)

## Theme ของรอบ

**4 พฤษภาคม คือวันที่ frontier lab ทุกค่ายยอมรับว่า model ไม่พอ — services delivery คือ choke point** ในวันเดียวกัน OpenAI ปิด **The Deployment Company** $10B JV กับ TPG/Brookfield/Bain (พร้อม return guarantee 17.5%/ปี โครงสร้างไม่เคยมีใน VC) — และ Anthropic ตั้ง standalone services firm $1.5B กับ Blackstone, Hellman & Friedman, Goldman Sachs ทั้งสองดีลใช้ **Palantir-style forward-deployed engineer** — ฝัง engineer ของตัวเองเข้า portfolio company ของ PE แล้ว rebuild workflow

วันเดียวกัน Sierra ของ Bret Taylor (OpenAI chairman) ปิด Series E $950M @ $15.8B — proof point ว่า **vertical AI agent product ใน CX สามารถแตะ $150M ARR ใน 8 ไตรมาส** ที่ traditional SaaS ไม่เคยทำได้ และเช้าวันเดียวกัน Abhishek Das ของ Yutori ออก essay บอกว่า **agent industry ทั้งหมด กำลัง normalize unreliability** ที่ math ของ compounding error จะทำให้ deployment fail — 90% accuracy ต่อ step → 35% success ใน 10-step workflow → 0.5% ใน 50-step

Pattern ที่ run ผ่านทั้งสี่เรื่อง: **infrastructure + delivery layer กำลัง capture value มากกว่า model layer** — และผู้แพ้ที่ชัดสุดคือ Big 4 consulting + BPO + middleware ที่กลางทาง
