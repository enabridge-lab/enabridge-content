- **Vertical depth ชนะ horizontal breadth ในตลาด enterprise** — Anthropic 10 banking agent + Microsoft 365 + Moody's + Dimon ขึ้นเวทีคู่; Sierra ขาย customer agent ลึกเข้า insurance/mortgage/healthcare; ทั้งคู่ใช้ playbook เหมือน Salesforce 2002 ที่ build vertical first OpenBridge ต้องเลือก 1-2 vertical Thai/SEA (banking + insurance) แล้ว build connector pack + agent template เฉพาะ — แทนการขาย "MCP server ทั่วไป"
- **Voice กลายเป็น default channel ของ customer-facing agent ไม่ใช่ feature เสริม** — GPT-Realtime-2 cached input $0.40/M = 50x ถูกกว่า uncached + translate $0.034/นาที + Whisper streaming $0.017/นาที + Sierra report voice แซง text ตั้งแต่ ต.ค. 2025; ตลาด BPO multi-lingual ของ SEA ($2/ชม translate) เปิด market ใหม่; OpenBridge ต้อง position MCP server ที่ "voice-aware" + carry context ระหว่าง turn เพราะ frontier voice model ไม่มี business state ของลูกค้า
- **Agent vertical SaaS เป็น category จริง — pricing power 100x ARR** — Sierra $15.8B / $150M ARR; round size 6x category average; 0-to-$150M ใน 7 ไตรมาส; GV ลีดร่วมบอกว่า Google มอง Sierra เป็นลูกค้าไม่ใช่คู่แข่ง — frontier vendor accept ว่า application-layer vertical SaaS เป็น customer; OpenBridge ต้องสร้าง **"powering AI agent platforms"** sales motion แยก: pricing per-event + revenue share จาก usage ของลูกค้าปลายทาง
- Action ตรงสำหรับ OpenBridge: (1) **Connector ทุกตัวต้อง emit Outcomes-compatible eval signal** — รายงาน success criteria ระดับ business outcome (เอกสาร reconciled กับ ledger, KYC ผ่าน policy) ไม่ใช่ HTTP 200 (2) **Vertical Thai/SEA banking + insurance pack** — connector + agent template + audit trail สำหรับ Guidewire/Encompass/Workday/Workiva (3) **Voice-aware MCP payload** — caller history + recent ticket + sentiment ที่ GPT-Realtime-2 / Claude voice ดึงผ่าน MCP เพื่อ ground voice agent ใน real customer state (4) **"Powering agent platforms" sales motion** — ขาย connector + audit trail ให้ Sierra/Decagon/Cresta/Parloa pricing per-event + revenue share (5) **Local data exclusive partnership** — คุย SET, SCBX, KBank, NSO, Wisesight สำหรับ Moody's-style integration ที่ frontier vendor ต้องผ่าน OpenBridge เพื่อแตะ data ใน SEA
