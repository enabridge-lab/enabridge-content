# Daily Brief — 26-05-18 รอบ 06:03 (Bangkok)

## Theme ของรอบ

**สัปดาห์ 11–17 พ.ค. 2026 = "Agentic economy เริ่มมี stack ครบ — deployment + payment + governance + threat"**. 4 ข่าวรอบนี้ประกอบกันเป็น mental model ของ agentic economy ที่ใช้งานจริง ไม่ใช่ keynote slides. ตั้งแต่ระดับ end-user deployment (Sierra ที่ 40% Fortune 50, voice แซง text เป็น primary channel), ลงไปถึง financial substrate (Circle Agent Stack ที่เปิด nanopayment ระดับ $0.000001 + agent wallet + marketplace), ลงลึกถึง regulated industry production (Broadridge ที่ 80% ของ Tier-1 banks ใช้อยู่ deploy agent ระดับ trade settlement), จบที่ threat surface (Google Security เปิด data IPI ที่ scan 2-3B page/month เจอ +32% growth ของ malicious injection)

**Sierra ($950M raise, $15.8B valuation)** เป็น signal ชัดที่สุดว่า voice agent ไม่ใช่ niche อีกต่อไป — ARR $150M ใน 3 ปี (เร็วที่สุดในประวัติศาสตร์ enterprise SaaS), customer 40%+ Fortune 50, proof point ที่ verify ได้: Nordstrom 5 สัปดาห์ deploy, Singtel 10 สัปดาห์ ที่ resolution rate >70%, Cigna 8 สัปดาห์ ตัด authentication time 80%. ที่สำคัญ — **voice แซง text เป็น primary channel ตั้งแต่ ก.ย. 2025** สะท้อนว่า enterprise demand เข้า phone automation เต็มตัว. valuation $15.8B = ~100x ARR (สูงกว่า peak SaaS multiple ยุค 2021 5-10 เท่า) แปลว่านักลงทุนเดิมพันว่า Sierra จะกินตลาด CCaaS $85B + ส่วน BPO $300B ในระยะยาว

**Circle เปิด Agent Stack** (11 พ.ค.) คือสิ่งที่ครบ-puzzle ของ agentic economy. MCP/A2A ทำให้ agent คุยกันได้แล้ว แต่ **เงินยังไม่ไหล** ระหว่าง agent. Circle Agent Wallet (policy-controlled, autonomous) + Agent Marketplace (agent browse + เลือก + จ่ายเอง) + Nanopayments ที่ระดับ **$0.000001** ผ่าน Circle Gateway = unlock economic layer ที่ Visa/ACH/SWIFT รองรับไม่ได้ (เพราะออกแบบสำหรับ human + institution). agent ที่ทำ inference proxying 10,000 calls/hr ที่ $0.0001/call ต้องการ rail ใหม่. Circle เดิมพันว่า USDC = native monetary layer + ตัวเองเป็น "App Store" ของ agent service

**Broadridge** (11 พ.ค.) เป็นเคสที่ pivot narrative จาก "agent เป็น nice-to-have" สู่ **"agent เข้า heart ของ regulated production"**. Broadridge อยู่ใน critical path ของ trade settlement ระดับ **$11 trillion/วัน** + 80% ของ Tier-1 banks ใช้ — risk threshold สูงสุดในโลก. agentic capability ที่ live ใน production: trade fail/break resolution, account opening, real-time valuation exception, customer inquiry, email workflow (กับ DeepSee เป็น AI-native partner). proof points: 40+ client ตั้งแต่ 2024, millions transactions/mo, **30% Day-1 operational cost reduction** สำหรับลูกค้าใหม่. ถ้าตัวเลขนี้ holds ใน 6 เดือน, ทุก buy-side/sell-side firm จะเปลี่ยน procurement จาก "explore AI" เป็น "rip-and-replace ops infra"

**แต่ระหว่างที่ทุกคนเร่ง deploy — Google Security ปล่อย counter-data ที่ enterprise CIO ต้องอ่าน**. Indirect Prompt Injection (IPI) คือ vector ที่ใส่คำสั่งซ่อนใน HTML รอ AI agent scrape page แล้ว execute ทันที. Google scan 2-3B page/mo เจอ malicious IPI โต **+32% ระหว่าง พ.ย. 2025 ถึง ก.พ. 2026** — payload ที่ documented: **คำสั่ง PayPal transaction เต็มรูปแบบ** + meta tag namespace injection ที่ใช้ keyword "ultrathink" route financial action ไป Stripe donation link. จุดเจ็บ: AI agent มี legitimate credential + service account ที่ approved → execute malicious command มองไม่ออกจากการทำงานปกติ, audit log ปกติจับไม่ได้. **82% executive เชื่อว่า policy ป้องกันได้, แต่จริง ๆ มีแค่ 37-40% ที่มี containment control** — ส่วนใหญ่ "เห็น" agent ทำงาน แต่ "หยุด" ไม่ได้

Pattern หลักประจำรอบ — **agentic economy โตเร็วในแนวขนาน 4 stack** (deployment/payment/regulated production/security). Sierra + Circle + Broadridge ทุกตัวคือ proof point ของ "production-ready" agent ในเลเยอร์ที่ต่างกัน. Google IPI คือ reminder ว่า attack surface โตเร็วกว่า defense playbook. ผู้ชนะของรอบนี้ — บริษัทที่ build agent platform ที่มี **enforcement point ตั้งแต่ day 1 (ไม่ใช่ monitoring แต่ containment)** + รองรับ pay-per-call ระดับ sub-cent + connector library ที่ certified กับ regulated industry. OpenBridge อยู่ในตำแหน่งที่ดีในการเป็นทั้ง 3 อย่าง
