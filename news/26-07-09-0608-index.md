## Theme ของรอบ

Signal: **เกม data ยุคหน้าไม่ใช่ scrape internet, แต่คือ synthesize บริษัท**

Thailand จะไม่รอด pattern นี้ใน H2 2026

Pattern หลัก 4 ข่าวรวมกัน — **stack agent ยุค H2 2026 = 4 ชั้นที่ vendor กำลังลง capital กันจริงจัง**: (1) **Training environment layer** (Bespoke, Patronus, METR); (2) **Runtime / sub-agent orchestration layer** ที่ frontier lab เอาไปทำเอง (OpenAI Ultra, Anthropic Sub-Agent, Gemini A2A); (3) **Sovereign distribution layer** (nation-state deal ระดับ $B); (4) **Vertical deterministic-agentic hybrid layer** (Anaplan, Snowflake, Salesforce). **ที่น่าคิดสำหรับตลาดไทย = ตำแหน่งของ builder/business ในโครงสร้าง 4 ชั้นนี้จะกำหนดกำไรใน 2027-2028**

## TL;DR

- **Bespoke Labs $40M = "training environment เป็น moat ใหม่ของ agent"**. Frontier lab จะจ่ายเพื่อ **synthetic company** ที่ agent วิ่งฝึก long-horizon เพราะ post-training บน real corporate data ทำไม่ได้ด้วยเหตุผล compliance. **Builders ในไทย**: ถ้าคุณ ship agent framework โดยไม่มี evaluation report จาก third-party environment (Bespoke, Patronus, METR) — RFP ครึ่งหลัง 2026 ตอบไม่ได้. **Business**: ถามทุก vendor ว่า **agent stamina ตลอด 8 ชั่วโมงเสถียรกี่ %** — vendor ที่ตอบไม่ได้ = คุณเป็น human evaluator ให้เขาโดยไม่รู้ตัว

- **OpenAI Ultra Mode 91.9% Terminal-Bench 2.1 = sub-agent orchestration ย้ายจาก app → runtime**. Pattern เดียวกับ Anthropic Claude Sub-Agent, Gemini A2A native = **frontier lab eat orchestration middleware**. LangChain/LangGraph/CrewAI/AutoGen moat ต้องย้ายไป (a) multi-vendor routing, (b) governance/observability, (c) domain-specific sub-agent library. **Business ในไทย**: ก่อนต่อสัญญา 1 ปี กับ orchestration vendor, ทดสอบ Sol Ultra ใน workflow เดิม — บาง case ประหยัด engineering effort 3-6 เดือน + latency ต่ำกว่า

- **Salesforce $1B Switzerland + Benioff co-chair Global AI Commission = agentic AI กลายเป็น sovereign infrastructure**. Pattern **"vendor commit $B ให้ nation-state"** ต่อเนื่อง — MSFT-UAE, OpenAI-Saudi, GOOG-India, ANTH-UK, ตอนนี้ CRM-Switzerland. **Thailand ควรเจรจาก่อน vendor commit $B ในไตรมาสหน้า**: (1) National AI Act ที่บังคับ data residency in-country ไม่ใช่ region; (2) audit trail ที่ regulator เข้าถึง direct; (3) sub-processor list ที่ vendor commit 90 วันก่อนเปลี่ยน. **Builders ในไทย**: startup ที่ pitch **local alternative + compliance-by-design** (PDPA/BOT/SEC/กสทช.) มี tailwind — vendor ใหญ่ไม่ compete เพราะ scale ไม่ถึง $B

- **Anaplan Agentic Enterprise = "deterministic + agentic hybrid" สำหรับ regulated vertical**. LLM-only ทำงาน CFO ที่ต้อง idempotent + audit trail ไม่ได้ — Anaplan รวม planning math engine + Bedrock agent layer + skill catalog 3 tier (automation/augmentation/advisory). Ship Oct 2026 ทันไตรมาส 4 CFO ทั่วโลก plan งบ 2027. **CFO ในไทย**: SET50/EGCO/PTT/SCB/KTB/CPALL/HMPRO ที่ใช้ Anaplan → คุย AE ขอ preview + pilot Q4; ที่ไม่ใช้ → benchmark กับ Workday Illuminate + Oracle Fusion AI ใน Q3 ก่อนวางสัญญา

- **Pattern หลักของ 4 ข่าวรวมกัน — stack agent H2 2026 = 4 layer ที่ vendor กำลังลง capital**: (1) **Training environment** (Bespoke, Patronus, METR); (2) **Runtime / sub-agent orchestration** (OpenAI Ultra, Anthropic Sub-Agent, Gemini A2A); (3) **Sovereign distribution** (nation-state $B deal); (4) **Vertical deterministic-agentic hybrid** (Anaplan, Snowflake, Salesforce Atlas). **ตำแหน่งของ builder/business ในโครงสร้าง 4 ชั้นนี้จะกำหนดกำไรใน 2027-2028**

  - (2) **Sub-agent orchestration ที่แค่ wrap LLM = commodity** — moat ต้องย้ายไป multi-vendor + governance + domain-specific
  - (3) **Sovereign compliance layer** (PDPA, ISO 42001, data residency) เป็น distribution advantage ในไทย/ASEAN ที่ vendor $B-scale ไม่ compete
