# Emergent เข้า unicorn club — $130M Series C ที่ $1.5B valuation หนึ่งปีหลัง launch สะท้อน agentic coding กำลังชนะ vibe-code ทั่วโลก

## TL;DR
- **15 กรกฎาคม 2026** — Emergent, agentic AI coding startup จาก Bangalore, ปิด Series C $130M ที่ post-money valuation $1.5B — โต 5 เท่าในหกเดือน
- Metric ใช้ได้: **$120M ARR run-rate**, **200,000 paying customers**, **12 ล้าน app** สร้างบน platform ตั้งแต่เปิดตัวปีที่แล้ว
- นำโดย Creaegis (PE ไทย-เจ้าเก่าของ ChrysCapital) พร้อม Khosla, SoftBank Vision Fund 2, Lightspeed, Y Combinator lean-in — total funding $230M

## เกิดอะไรขึ้น

Emergent — บริษัทที่ก่อตั้งปี 2024 โดยทีมอินเดีย เจาะกลุ่ม non-technical founder ให้สร้าง app ด้วย natural-language prompt — เพิ่งประกาศ Series C $130 ล้านที่ post-money valuation $1.5 พันล้านเมื่อวันที่ 15 กรกฎาคม. หกเดือนก่อนหน้าที่ Series B, valuation อยู่ที่ราว $300M — โต 5 เท่าในสองไตรมาส. ผู้นำรอบใหม่คือ Creaegis, PE firm ที่แยกออกมาจาก ChrysCapital ในปีที่แล้ว. Khosla Ventures, SoftBank Vision Fund 2, Lightspeed India, และ Y Combinator ลง follow-on แทบทุกกอง — signal ว่าไม่มี insider ยอมให้ dilution เอง

ตัวเลขที่ Emergent เปิดคือ $120M annualized revenue run-rate กับ 200,000 paying customers และ 12 ล้าน app ที่ user สร้างบน platform. อัตราส่วน ARR ต่อ valuation ราว 12.5x — ต่ำกว่า Cursor / Replit / Bolt ที่ระดับ 20–40x หมายความว่ารอบนี้ราคา "ถูก" ตามมาตรฐาน agentic coding — หรือแปลว่านักลงทุนเชื่อว่ายังโตต่อไปได้เร็วมาก

หัวใจของ product คือ agent stack ที่ pipeline หลาย model ต่อกัน — planner หนึ่งตัวย่อย spec, code-generator อีกตัวเขียน component, verifier อีกตัว sanity-check output แล้ววน iteration จนได้ deployable app. เทียบกับ Bolt / Lovable ที่ยัง "vibe-code single-shot", Emergent claim ว่า success rate ของ complex app (ที่ต้องมี DB + auth + payment) สูงกว่าประมาณ 2 เท่า — ตัวเลขที่ยังไม่มี third-party ยืนยัน แต่ 200K paying customer เป็นตัวบ่งชี้ทางอ้อมที่มีน้ำหนัก. Emergent CEO บอกว่าจะเอาเงินไปลงสองด้าน — improving success rate ของ core agent workflow และขยาย go-to-market ออกนอกอินเดีย (ปัจจุบัน user เกินครึ่งอยู่ US)

## ทำไมสำคัญ

pattern สำคัญคือ **agentic coding platform ตัวแรกจากอินเดียเข้ายูนิคอร์นเร็วกว่าที่ตลาดคาด**. ปีที่แล้วเมื่อ Emergent launch หลายคน dismiss ว่าเป็น regional Bolt-clone. หนึ่งปีต่อมามี $120M ARR — ใกล้เคียงกับ Bolt.new ตอนที่ปิด Series B ($200M ARR) และเกินหลาย coding tool ที่ตั้งอยู่ Silicon Valley. Signal คือ agentic coding เป็น market ที่ **product ชนะไม่ใช่ location** — user ทั่วโลกเลือก tool ที่ success rate สูงและ latency ต่ำโดยไม่สนใจ vendor country

pattern ที่สองคือ **$1.5B ยังเป็นราคาถูกสำหรับ tier-1 agentic tool**. Cursor $10B (Q1 ปีนี้), Cognition $5B, Replit ล่าสุด $3.5B. ถ้า Emergent รักษาการเติบโตแบบ 5x ในหกเดือนต่อไป Series D กลางปีหน้าน่าจะแตะ $4–5B — และนั่นจะทำให้ India กับ US มี "big four" ของ agentic coding platform (Cursor, Cognition, Replit, Emergent) โดยไม่มี European หรือ Chinese player ในระดับเดียวกัน

signal ต่อไปที่น่าจับตา: (1) enterprise SKU — 200K paying customer ปัจจุบันเน้น individual / SMB, ถ้า Emergent แตะ Fortune 500 pipeline ได้จะเปลี่ยน narrative จาก "consumer no-code" เป็น "enterprise agent". (2) Model dependency — ยังไม่ชัดว่า Emergent ใช้ base model ของใคร; ถ้าเป็น Anthropic / OpenAI ทั้งหมด gross margin จะกินโดย inference cost เมื่อ scale. (3) India stack effect — ถ้า Emergent สำเร็จ, Sarvam, Krutrim, และ Nyun จะได้ narrative tailwind และดึง talent กลับอินเดียจาก Silicon Valley

## มุม AI Agent Platform

**Builders**: agentic coding พิสูจน์แล้วว่าเป็น pattern ที่ agent multi-step ชนะ vibe-code single-shot ในตลาดจริง — ถ้าคุณสร้าง agent framework ให้ vertical อื่น (legal, finance, ops) pattern เดียวกันน่าจะเป็นสถาปัตยกรรมชนะ. **Users / business** — ถ้าองค์กรคุณยังไม่ audit ว่าทีม product / engineering ใช้ AI coding tool อะไรอยู่ วันนี้ควรทำ อย่างน้อยต้องรู้ว่า code base มี AI-generated portion กี่เปอร์เซ็นต์ กระบวนการ review / security scan รองรับหรือยัง. **Ecosystem** — GitHub Copilot กับ Amazon Q Developer เริ่มเสีย share ให้ agent-native tool อย่างเห็นได้ชัด; Microsoft ต้อง repositioning Copilot Workspace เร็วขึ้น. สำหรับ Enabridge — บริษัทไทยที่ต้องการสร้าง MVP รอบใหม่ ทดลอง deploy Emergent หรือ agentic coding tool เทียบกับ Bolt / Lovable เพื่อดู success rate และ cost per feature เป็นการบ้านที่ควรทำสัปดาห์นี้
