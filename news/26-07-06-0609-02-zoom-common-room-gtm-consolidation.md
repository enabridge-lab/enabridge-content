# Zoom ซื้อ Common Room — GTM AI-native ปิดรอบที่ 3 ใน 8 เดือน หลัง Clari-Salesloft และ Apollo-Pocus

## TL;DR
- Zoom Communications เซ็น **definitive agreement ซื้อ Common Room** (2 ก.ค.) — AI-native GTM intelligence platform ที่รวม signal จาก CRM + product + marketing + engagement + real-world buying signals แล้วขับ **RoomieAI agents** ทำ account research / message personalization / prospecting
- Common Room ใช้อยู่ที่ **Atlassian, Anthropic, Autodesk, Notion, Okta, Snowflake** — ระดับ B2B tech tier-1
- **Third GTM-intelligence consolidation ใน 8 เดือน** — Clari ซื้อ Salesloft (พ.ย. 2025), Apollo ซื้อ Pocus (มี.ค. 2026), Zoom ซื้อ Common Room (ก.ค. 2026). Financial terms ทั้ง 3 deal ไม่เปิดเผย — สัญญาณของ pattern เดียวกัน: **AI-native revenue stack ปิดรวมเป็น 3 เจ้า**

## เกิดอะไรขึ้น
เช้าวันพฤหัสฯ 2 กรกฎาคม 2026 Zoom Communications ประกาศเซ็น definitive agreement ซื้อ **Common Room** — GTM intelligence platform อายุ 5 ปีที่รวม first-party data ข้าม CRM / product / marketing / engagement systems + buying signals จากโลกจริง (community, event, code repo, technical discussion) → ส่งเข้า **RoomieAI agents** ที่ทำ **account & contact research → message personalization → prospecting** ให้ SDR/AE ก่อนสายเรียก. Financial terms ไม่เปิดเผย

Zoom framing เป็น "extension ของ Zoom Revenue Accelerator" (product ที่ record + analyze sales conversation เพื่อ real-time coaching + deal intel + forecast). สูตร post-deal ที่ Zoom วางไว้: **buying signal จาก Common Room → prep call → Zoom video meeting → post-call intel + coach → forecast update = closed loop signal-to-close ในแพลตฟอร์มเดียว**. ลูกค้าปัจจุบันของ Common Room ที่ Zoom เอามาด้วย: **Atlassian, Anthropic, Autodesk, Notion, Okta, Snowflake** — คุณภาพ customer ระดับ B2B tech tier-1 ที่ HubSpot/Salesforce จะฟาดเข้า EDP ยาก

Deal นี้ **ครบสาม** ของ GTM-intelligence consolidation ใน 8 เดือน:
- **Clari ซื้อ Salesloft** — พฤศจิกายน 2025; รวม revenue orchestration + sales engagement
- **Apollo ซื้อ Pocus** — มีนาคม 2026; รวม outbound prospecting + product-led signals
- **Zoom ซื้อ Common Room** — กรกฎาคม 2026; รวม video/audio conversation + AI-native GTM intelligence

ทั้ง 3 deal ไม่เปิด financial term — สัญญาณว่า valuation ยังอยู่ในระดับที่ acquirer ไม่อยาก set market comp. Common Room ระดม Series B $52M ที่ post-money ~$500M ปลาย 2024; ประมาณการวงในว่า Zoom จ่ายในระดับ $700M–$1.2B ผสม cash + Zoom stock

## ทำไมสำคัญ
คำถามที่ Wall Street ถามตลอด 12 เดือน = **"Zoom จะ pivot ยังไงในยุค Teams + Google Meet + AI note-taking commoditize"**. Zoom Revenue Accelerator เป็นคำตอบเดียวที่ CFO Michelle Chang ให้ตลอด — แต่มี blind spot ตรง **pre-call**: SDR/AE ยังไม่รู้ว่า customer มี signal อะไร trigger ให้เรียก. **Common Room เติมช่องนั้นตรง ๆ** — RoomieAI ทำ account research + write outbound message + prioritize deal ก่อน sales rep เข้าประชุม → เข้าประชุมใน Zoom → post-call insight ไหลกลับ Common Room → next-action loop

**Pattern ของ 3 deal รวมกัน**: GTM stack ที่แต่ก่อนซื้อแยก 4–7 tool (Salesforce, Outreach, Gong, Clari, ZoomInfo, Apollo, Common Room, Salesloft) กำลังรวมเป็น **3 กลุ่มหลัก**:
1. **Clari + Salesloft** — revenue orchestration + sales engagement
2. **Apollo + Pocus** — outbound + PLG signals
3. **Zoom + Common Room** — conversation + buyer intel + video meeting

Salesforce (Data Cloud + Agentforce SDR/BDR) + HubSpot (Breeze + Content Hub) + Microsoft (Sales Copilot + Teams) เป็น incumbent ที่ต้อง response — คาดว่าอีก 2 ไตรมาสน่าจะเห็น **Salesforce ซื้อ Gong** หรือ **HubSpot ซื้อ Common Room-like** เพื่อ close signal-to-close loop

Signal ที่กว้างกว่า: **AI-native revenue tool ที่ raise $20M–$100M ในช่วง 2022–2024 กำลังโดนกิน** โดย big platform ที่มี **distribution + billing + integration**. Standalone signal tool (Champion tracking, buyer intelligence, product-usage-to-outbound signal) 12 เดือนข้างหน้าจะเลือก: (1) ขายให้ platform, (2) becoming platform ตัวเอง — ทางที่ 3 (แข่งกับ platform) โดน bypass เร็ว

## มุม AI Agent Platform
- **Builders** — startup ที่สร้าง agent สำหรับ sales/marketing/GTM ควรถามคำถามเดียว: **"integration surface ของเราวิ่งบน stack ของใครใน 3 กลุ่มนี้"**. ถ้าตอบไม่ได้ = risk สูง 12 เดือน. **RoomieAI-clone (agent ทำ account research + outreach automation)** มี window แคบมาก — Common Room เพิ่งโดนซื้อ = market signal ว่า acquirer พร้อมจ่าย. Framework ที่ agent SDR/BDR-first (LangGraph supervisor + tool use + browser) มี tailwind แต่ต้อง embed ใน Zoom/Salesforce/HubSpot ecosystem ไม่ใช่ SaaS แยก. **Vertical GTM agent** (industry-specific: healthcare sales, banking wealth advisor, industrial CPG rep) ยังปลอดภัยจากการโดน bundle — big platform ยังไม่ verticalize

- **Users/business ในไทย** — ทีม sales/marketing ในไทยที่ใช้ Zoom + Salesforce (SCB, KBank, KTC, AIS, True, Central, CP) — จับตา RoomieAI integration timeline เข้าไทย 6–9 เดือนหลัง GA US, **ให้เตรียม procurement + data residency review** (PDPA + first-party data pipeline) ตั้งแต่ตอนนี้. บริษัทที่ใช้ Common Room อยู่ = 0 องค์กรในไทย, แต่ **pattern signal-to-close ที่ Zoom sell** จะเข้ามาผ่าน APAC channel ของ Zoom + Salesforce ไตรมาส Q4 2026 – Q1 2027. บริษัทที่ยังใช้ Sales Navigator + Excel + Salesforce แบบ manual = **RFP รอบต่อไปควรบวก "AI-native buyer intelligence" เป็น line item**

- **Ecosystem** — CDP (Segment, mParticle, Rudderstack) กำลัง compete หนักกับ **AI-native signal orchestrator** (Common Room, Champion, UserGems) — หลัง deal นี้ CDP ต้อง add **agentic layer** ไม่งั้นเป็นแค่ pipe. **Content generation tool** (Jasper, Copy AI, Writer) → merged ใน platform นี้ในระดับ tactical output. **APAC/SEA channel**: Zoom APAC + Salesforce ASEAN + Microsoft ASEAN — บริษัทที่มี pipeline gap ระหว่าง signal → outreach ก็มี catalyst จริงจังใน 12 เดือน
