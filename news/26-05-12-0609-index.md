# Daily Brief — 26-05-12 รอบ 06:09 (Bangkok)

## Theme ของรอบ

**24-48 ชม. ที่ผ่านมาคือ "services era week" ของ frontier AI** — สี่ข่าวใหญ่บอกตรงกันว่า war ของ enterprise AI กำลังขยับจาก "model API + connector" เป็น **"engineer ที่ฝังเข้าไป redesign workflow ลูกค้า"** วันที่ 11 พ.ค. OpenAI ตั้ง **OpenAI Deployment Company** initial fund **$4B จาก 19 partners** — lead **TPG**, co-lead **Advent / Bain Capital / Brookfield**, founding partner **BBVA / Goldman Sachs / SoftBank / Warburg Pincus / WCAS / Emergence / B Capital / Goanna** วันเดียวกันซื้อ **Tomoro** บริษัท applied AI consulting จาก London ดึง **~150 Forward Deployed Engineers** เข้าวันแรก — valuation รวม **$14B** การที่ Goldman + BBVA ลงทุนเอง signal ว่า bank ต้องการ deployment capability ฝังตรงข้างใน ไม่ใช่ outsource ให้ Accenture/IBM ต่อ — OpenAI ยอมแข่งกับ services giant ที่เคยเก็บ margin 3-7x ของ software ARR

วันก่อนหน้า (4 พ.ค.) Anthropic ทำ pattern เดียวกันแล้วกับ **FIS** ผู้ให้บริการ core banking ระดับ global (รายได้ $10B/ปี + เป็น core ของ 9 ใน 10 bank ใหญ่สุดของสหรัฐ) เปิด **Financial Crimes AI Agent** ตัวแรก agent ที่ run on **Claude Opus 4.7 financial-tuned** บีบ AML investigation จาก **2-5 วัน เหลือ 8-12 นาที** (~250x speedup), false positive rate ลด **~60%** AML/compliance market มี cost ~**$40B/ปี** — saving potential $12-16B ต่อปี **BMO** (asset $1.2T) + **Amalgamated Bank** (asset $8B) เป็น lighthouse customer **Anthropic Applied AI team + FDE** ฝังตรงกับ FIS เพื่อ co-design + knowledge transfer ให้ FIS build agent ถัดไปเอง — template เดียวกับที่ OpenAI scale ขึ้น 7 วันให้หลัง

ขณะที่ทุก vendor วิ่งเข้า services Anthropic ก็จุดวิกฤต cyber ทั้งวงการ — model **Mythos** ที่ปล่อยผ่าน **Project Glasswing** controlled rollout (~40 องค์กร) พบ **271 zero-day** ใน Firefox ก่อน release v150 (เทียบ model รุ่นก่อนที่หาได้แค่ 20); รวม catalog ทุก software อยู่ระดับ **tens of thousands vulnerabilities** Dario Amodei เตือนมี **window 6-12 เดือน** ที่ defender ต้อง patch ก่อน Chinese AI catch up; **Fed Chair Powell + Treasury Sec Bessent** เรียก CEO bank ใหญ่ประชุม cyber risk; **Jamie Dimon** เรียก "very heightened risk to financial system"; **IMF** ออก formal advisory นี่คือ **first prove case** ที่ AI เปลี่ยน threat landscape ระดับ macro — และเป็น geopolitical lever ที่ Anthropic ใช้ build trust กับ US government เร็วกว่า OpenAI/Microsoft/Google

ปิดสัปดาห์ด้วย cautionary tale: CTO ของ **Uber** ยอมรับใช้ **งบ AI ทั้งปี 2026 ($3.4B) หมดใน 4 เดือน** — Claude Code + Cursor adoption ระเบิด: **monthly cost per engineer $500-$2,000**, **95% engineer ใช้ AI ทุกเดือน**, **70% ของ committed code มาจาก AI** ตัว engine คือ **Minions** internal coding agent ที่ produce **~1,800 change/week zero human authoring** (8% ของ all code change ของ Uber) Root cause ไม่ใช่ราคา Claude แพง — แต่เป็น **organizational gap** ระหว่างทีมที่ขับ adoption กับทีมที่ manage spend Q3-Q4 จะเห็น Goldman/JPMorgan/Disney/Netflix เจอ pattern เดียวกัน — และตลาด **AI FinOps** จะเปิดเป็น category ใหม่

Pattern หลักประจำสัปดาห์: **services + cybersecurity + cost governance** กลายเป็น sub-market ใหม่ที่ใหญ่ระดับหลายร้อยพันล้านดอลลาร์ ภายใน 12 เดือน FDE จะเป็น decision maker คนหนึ่งในทุก enterprise AI deal, vendor MCP/connector ทุกตัวต้อง Mythos-cleared + tamper-proof audit, และทุก enterprise ที่ deploy Claude Code/Cursor จะหา token budget governance OpenBridge ต้อง position ที่ **"infrastructure ที่ FDE ของทุก vendor ใช้ + Mythos-cleared + finance-friendly"** — ไม่ใช่ alternative ของ FDE, ไม่ใช่ pure-play security vendor, ไม่ใช่ AI FinOps startup แต่เป็น horizontal layer ใต้ทุก vertical agent ที่ขายใน SEA
