# Daily Brief — 26-05-20 รอบ 06:04 (Bangkok)

## Theme ของรอบ

**สัปดาห์ 18-19 พ.ค. 2026 = "Vendor war ของ enterprise AI ขยับเข้าชั้น infrastructure"**. 4 ข่าวรอบนี้ประกอบกันเป็น mental model ของ enterprise AI stack ที่กำลัง re-architect — ทั้ง ค่าย OpenAI กับ Anthropic ที่เลือก philosophy ตรงข้ามแก้โจทย์ data residency, vertical agent (Moment) ที่ proof ว่า incumbent ยอม rip-and-replace, และ challenger (Dust) ที่ challenge thesis "single-player AI" ของทั้ง ChatGPT + Copilot

**Anthropic MCP Tunnels + Self-Hosted Sandboxes** (19 พ.ค. ที่ Code with Claude ลอนดอน) เป็น signal ใหญ่ที่สุดของรอบ — Anthropic ปลดล็อก agent ให้เข้า private network ของลูกค้าโดยไม่ต้องเปิด public endpoint, ทำงานคู่กับ self-hosted sandbox ที่ย้าย tool execution ไปฝั่งลูกค้า (รันบน Cloudflare/Daytona/Modal/Vercel ก็ได้). Architecture แยกหน้าที่ชัด — orchestration loop อยู่ที่ Anthropic, tool execution + data access อยู่ฝั่งลูกค้า. แก้โจทย์ใหญ่ที่สุดของ Tier-1 enterprise procurement (data residency + network isolation) ที่ทำให้ bank + insurance + healthcare ยอม sign deal Claude

**OpenAI x Dell** (18 พ.ค.) คือ counter-architecture. แทนที่จะ extend tentacle เข้า private network แบบ Anthropic — OpenAI ย้าย model + agent ลงไปอยู่บน Dell on-prem hardware (Dell AI Factory + Dell AI Data Platform). Codex ที่ตอนนี้มี 4 ล้าน dev/wk + ขยายออกจาก code ไป business workflow (lead qual, follow-up, feedback routing) จะ deploy ได้ใน datacenter ของลูกค้า. 2 ค่ายแก้โจทย์เดียวกันด้วย philosophy ตรงข้าม — **OpenAI bring model to data, Anthropic bring data path to model**. ใครชนะขึ้นกับ workload latency-sensitive แค่ไหน + compliance regime ของลูกค้า

**Moment** (19 พ.ค.) เป็น proof point ใหญ่ที่สุดของรอบว่า **agentic AI ใน regulated industry พ้น experimentation แล้ว — เข้า incumbent replacement**. Moment ที่อดีต quant Citadel ตั้งขึ้นปิด Series C **$78M** นำโดย Index Ventures + a16z + Avra. ตัวเลขที่ทำให้ดีลนี้พิเศษ — **ลูกค้า standardize บน platform คือ Edward Jones + LPL Financial + Hightower Advisors** รวม AUM **$10 ล้านล้าน** จาก $300 พันล้านเมื่อ 18 เดือนก่อน (33x growth). 3 agent core: portfolio construction (สร้าง portfolio จาก natural language ใน "วินาที"), surveillance (scan account หา tax/risk/transition opportunity), compliance (real-time scan vs custom rule)

**Dust** (18 พ.ค.) เป็น signal ว่า EU agentic AI champion ส่งมือถึงระดับโลก. Paris-based สตาร์ตอัพปิด Series B **$40M** นำโดย Sequoia + Abstract + Snowflake + Datadog. Production metric: **3,000 องค์กร, 51,000 MAU, 300,000 agent deploy, zero churn ตลอดปี 2025**. Thesis "multiplayer AI" — แทน assistant แบบ single-player (ChatGPT 1-1, Copilot per-user) ให้ทีม + agent ทำงานบน workspace เดียว, shared context, bidirectional notification. เป็น direct challenger ของ Glean + Copilot Studio + ChatGPT Enterprise — และ retention zero churn บอกว่า platform "stick" จริง

Pattern หลักประจำรอบ — **enterprise AI กำลังเปลี่ยน control point จาก "model" สู่ "infrastructure + workspace"**. Anthropic + OpenAI ไม่พอใจอยู่หลัง API — ต้องครอบ orchestration + connectivity + execution. Moment + Dust ขยับขึ้นไปครอบ workspace ของ end user — ทำให้ลูกค้าไม่ต้องคิดว่าใช้ model ไหน. ผู้ชนะของรอบนี้ — บริษัทที่ build **stack ที่ครอบ 3 ชั้น (model + infra + workspace) ก่อนคู่แข่ง**. OpenBridge ที่ขาย integration layer อยู่ในตำแหน่งที่ดีของชั้น infra แต่ต้องเร่ง strategy ว่าจะเป็น "vendor-neutral connector backbone" หรือ "embedded layer ใต้ Dust/Sierra/Glean"

## TL;DR สำหรับ Yoh

- **Anthropic MCP Tunnel = inflection point ของ MCP จาก spec สู่ enterprise infrastructure layer** — แก้กำแพงเดียวกันที่ทุก enterprise เจอ (internal MCP server เปิด public ไม่ได้, VPN ปวดหัว). Anthropic ทำ pattern เดียวกับ Cloudflare Tunnel ที่เคยทำให้ web — แต่ตอนนี้สำหรับ agent. ระยะ 6-12 เดือน เราจะเห็น **architectural divergence ของ enterprise AI 2 ค่าย**: OpenAI/Dell ย้าย model ไป on-prem (bring model to data), Anthropic เก็บ model ไว้แต่ extend tunnel ไป private network (bring data path to model). OpenBridge เจอ commoditization risk สูง — ถ้า MCP Tunnel ออก GA Q3 ลูกค้า Thai bank อาจไม่ต้อง buy connector layer แยก

- **Moment $78M + $10T AUM คือ proof point ใหญ่ที่สุดของรอบ** ว่า agentic AI ใน wealth management พ้น experimentation แล้ว — เข้า incumbent replacement. Edward Jones (50K advisor) + LPL Financial ($1.7T) + Hightower ($200B+) ยอมเปลี่ยน OS ทั้งบริษัท ไม่ใช่ pilot. AUM growth 33x ใน 18 เดือนคือ signal ว่า platform "stick" จริง. - **OpenAI x Dell คือ counter-narrative ที่ OpenAI ยอมรับ public ว่า cloud-only ไม่พอ** — เป็นครั้งแรกใน 2 ปี. - **Pattern หลักของ 4 ข่าวรวมกัน — control point ของ enterprise AI กำลังเปลี่ยน จาก "model" สู่ "infrastructure + workspace"**.
