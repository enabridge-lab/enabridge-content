# Sierra ระดม $950M ที่ valuation $15B — เปลี่ยน call center เป็น voice agent ที่ 40% ของ Fortune 50 ใช้แล้ว

## TL;DR
- 4 พ.ค. Sierra (Bret Taylor — อดีต co-CEO Salesforce + ประธานบอร์ด OpenAI) ปิดราวด์ $950M นำโดย Tiger Global + GV — valuation พุ่งเป็น **$15.8B** (จาก $10B เมื่อไม่กี่เดือนก่อน), เงินสดในแบงก์รวมเกิน $1B
- ARR แตะ $150M ในปีที่ 3 — ทะลุ $100M ARR ใน 7 ไตรมาส (เร็วที่สุดในประวัติศาสตร์ enterprise SaaS), ลูกค้าครอบคลุม **40%+ ของ Fortune 50**, voice แซง text เป็น primary channel ตั้งแต่ก.ย. 2025
- Proof points ตัวเลข: Nordstrom voice agent "Nora" deploy ใน **5 สัปดาห์**, Singtel **10 สัปดาห์** ที่ resolution rate >70%, Cigna **8 สัปดาห์** ตัด authentication time ลง **80%**

## เกิดอะไรขึ้น

วันที่ 4 พ.ค. 2026 Sierra — สตาร์ตอัพ customer service agent ของ Bret Taylor + Clay Bavor (อดีตหัวหน้า Google Labs) — ประกาศปิด funding round $950M ดันบริษัทขึ้นไปอยู่ที่ valuation **$15.8 พันล้านดอลลาร์**. ราวด์นี้นำโดย Tiger Global + GV โดยมี Sequoia, Benchmark และนักลงทุนเดิมทุกคนตามมาด้วย. เพิ่งผ่านราวด์ $350M ที่ $10B ไปไม่กี่เดือนก่อนหน้านี้ — แปลว่า valuation บวกขึ้น 58% ในไตรมาสเดียว, รวมเงินสดในแบงก์เกิน **$1 พันล้าน**

ตัวเลขที่ทำให้นักลงทุนเขย่า: Sierra เข้าปีที่ 3 ด้วย **ARR > $150M**, ทะลุ $100M ARR ใน 7 ไตรมาสหลังเปิดตัวเดือนก.พ. 2024 — pace ที่บริษัทเองเรียกว่า "เร็วที่สุดในประวัติศาสตร์ enterprise software". ลูกค้าตอนนี้ครอบคลุม **40% ของ Fortune 50** + agents ที่ build บน Sierra ทำงานครอบคลุม "พันล้านบทสนทนา" รวมถึง refinance บ้าน, ประมวลผล insurance claim, return order, fundraiser และ patient auth. ที่สำคัญที่สุด — **voice แซง text เป็น primary channel** ตั้งแต่ก.ย. 2025, สะท้อนว่า enterprise demand กระโดดเข้า phone automation เต็มตัว

Proof points ที่ Sierra เปิดเผยพร้อมราวด์ทำให้ดีลนี้แตกต่างจาก hype รอบอื่น. **Nordstrom** เปิดตัว voice agent "Nora" ภายใน **5 สัปดาห์**; **Singtel** (telco ใหญ่ที่สุดในเอเชียตะวันออกเฉียงใต้) deploy production ใน **10 สัปดาห์** ที่ resolution rate **>70%**; **Cigna** (หนึ่งใน health insurance ที่ใหญ่ที่สุดในโลก) เข้า production ใน **8 สัปดาห์** ตัด time-to-authenticate ของผู้ป่วยลง **80%**. ทั้งสามเคสคือเลข deployment ที่ verify ได้ — ไม่ใช่ "pilot ใน 50 stores" หรือ "PoC ใน 100 employees" ที่ vendor ส่วนใหญ่ใช้

## ทำไมสำคัญ

นี่คือ inflection point ของ **agentic AI ในตลาด customer service**. ก่อนหน้านี้ตลาดถูกแบ่งระหว่าง legacy CCaaS (Genesys, NICE, Five9) กับ low-cost chatbot ที่ resolution rate ไม่เกิน 30%. Sierra ที่ build ตั้งแต่วันแรกเป็น "agent-first" — ใช้ multi-model orchestration + voice-native + domain-specific tuning per customer — ทำให้ resolution rate กระโดดไปอยู่ระดับ 70-90% ใน 8-10 สัปดาห์. นั่นคือสิ่งที่ทำให้ Fortune 50 ยอม rip-and-replace stack เดิม

มอง 12-18 เดือนข้างหน้า — Sierra กำลังเปลี่ยนรูปแบบการแข่งของ enterprise software ทั้งเซกเตอร์. ARR ที่โต $100M → $150M ใน 6 เดือน (50%+ QoQ) ที่ valuation $15.8B หมายความว่า **trading multiple ~100x ARR** — สูงกว่า peak SaaS multiple ของยุค 2021 ถึง 5-10 เท่า. นักลงทุนกำลังเดิมพันว่า Sierra จะกินตลาด CCaaS ($85B globally) + ส่วนหนึ่งของ BPO ($300B) — รวม ~$400B addressable market. ถ้า capture แค่ 10% ก็ $40B revenue ซึ่งทำให้ $150M ARR วันนี้ดูเหมือนแค่จุดเริ่มต้น

อีกประเด็น — Sierra เป็น proof ว่า **vertical agent บน voice** เปลี่ยนสมการ unit economics ของ contact center. ลูกค้าที่ deploy ใน 5-10 สัปดาห์ + ตัด cost ลง 70%+ + เพิ่ม CSAT พร้อมกัน คือเคสที่ enterprise procurement หายากที่จะปฏิเสธ. ส่วนคู่แข่ง — Avoca ($1B HVAC voice), Vapi ($50M Amazon Ring), Cresta, Ada — กำลังเร่งหา niche ของตัวเอง แต่ Sierra cycle เร็วและกินเซกเตอร์กลางได้เร็วเกินกว่าใครจะตาม

## มุม OpenBridge

ตรงกระทบ. Sierra คือ proof point ว่า **voice agent ขายได้ใน scale**, แต่ Sierra เลือก vertical (customer service) — ไม่ใช่ horizontal integration. OpenBridge อยู่ในตำแหน่งที่ดีกว่าในการเป็น **integration backbone ของ Sierra-like agents** เพราะลูกค้า enterprise ที่ deploy voice agent ต้องต่อกับ CRM, billing, scheduling, knowledge base, identity layer — 8-15 ระบบต่อ deployment. ถ้า OpenBridge สร้าง **"voice-agent connector kit"** ที่ certify กับ Sierra (และคู่แข่ง Avoca/Vapi) — สามารถ position เป็น integration layer ที่อยู่ใต้ voice agent ทุกตัว ในตลาด SEA

อีก angle — Singtel deploy ใน 10 สัปดาห์ คือสัญญาณว่า **telco ในเอเชียตะวันออกเฉียงใต้** กำลังเปิดงบ voice agent. AIS, True, DTAC ในไทยและ Maxis, Celcom ใน MY ยังไม่มี Sierra-equivalent ในประเทศ. OpenBridge ที่มี connector library ที่ทำงานกับ Telco BSS/OSS อยู่แล้ว มี window 6-12 เดือนในการเป็น preferred integration partner ก่อนที่ Sierra หรือ Avoca จะตั้ง regional team
