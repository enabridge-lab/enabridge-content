**Pattern หลักของ 4 ข่าวรวมกัน** — **stack agent enterprise 2026 เห็นชั้นชัดมากขึ้น**: (1) **Runtime layer** = data platform (Databricks) หรือ hyperscaler (Azure/AWS) รวม governance + orchestration ในบ้านเดียว, (2) **Model layer** = subagent-native (Ultra mode, Sub-agent) กลายเป็น table stake, (3) **Portability layer** = MCP + Agent Skills เป็น open standard ที่คุมไม่ให้ enterprise lock-in ระดับ vendor, (4) **Distribution layer** = commerce / discovery ผ่าน AI assistant กลายเป็นช่องขายใหม่ที่ตัด aggregator เดิม. **ครึ่งหลัง 2026 = การประกอบ 4 ชั้นนี้เข้าด้วยกันใน production — เลือกวางชั้นให้ถูก, ไม่ใช่เลือก vendor**
