## Theme ของรอบ

**สัปดาห์ 14–15 พ.ค. 2026 คือ "Anthropic ขยับครอง enterprise GTM + research counter-signal" week** — PwC + Anthropic ประกาศ alliance ใหญ่ที่สุดของ Anthropic จนถึงตอนนี้: train และ certify พนักงาน PwC 30,000 คนเป็น Claude expert + ตั้ง Joint Center of Excellence + เปิด "Office of the CFO" เป็น standalone business unit แรกที่ build บน Claude เป็น foundation; โรลออก Claude Code + Cowork ที่ทีม US ก่อน ขยายสู่ workforce หลายแสนคน; proof point: insurance underwriting 10 สัปดาห์ → 10 วัน, security work ชั่วโมง → นาที, delivery time ลดสูงสุด 70%; flagship deployment คือ Advocate Health workforce 167,000 คน. Axios รายงานว่า move นี้ตามจังหวะที่ OpenAI court agent user หนักด้วย free tier เยอะกว่า — สัญญาณว่า frontier lab รายใหญ่ยอมแยก chat กับ agent เป็นสอง economic engine แล้ว ทุก agent product จะต้องผนวก cost observability + budget alarm เป็น first-class feature

แต่ระหว่างที่ vendor วิ่งขาย agent — **Microsoft Research ออก paper ตบหน้า narrative** ทั้งวงการ. ทีม Laban/Schnabel/Neville เผยแพร่ preprint "LLMs Corrupt Your Documents When You Delegate" ทดสอบ DELEGATE-52 benchmark ที่ครอบคลุม 52 vertical domain — Gemini 3.1 Pro, Claude 4.6 Opus, GPT 5.4 เสีย document content เฉลี่ย 25% หลัง 20 delegated interaction, เฉลี่ยทุก model degrade 50%, catastrophic corruption (<80%) เกิดในกว่า 80% ของ model/domain combination. จาก 52 vertical มีแค่ **Python programming** ที่ผ่าน 98% ready bar. จุดเจ็บที่สุด — เมื่อเพิ่ม agentic harness (file read/write + code execution tool) performance **แย่ลงเฉลี่ย 6 percentage point** — "ให้เครื่องมือเยอะขึ้น = พังเร็วขึ้น" สำหรับ workflow ที่ไม่ใช่ code

ฝั่ง startup — **Graphon AI** ออกจาก stealth ด้วย seed $8.3M (Novera Ventures นำ + Perplexity Fund, Samsung Next, GS Futures, Hitachi Ventures, Gaia, B37, Aurum — syndicate tilt ไป Asia + industrial). Positioning: **first pre-model intelligence layer** — ทำงานก่อน data จะเข้า context window ใช้ "graphon function" (คณิตศาสตร์ที่ Christian Borgs formalize ปี 2008) auto-discover ว่า data ใน enterprise เชื่อมโยงกันยังไง ครอบคลุม video, audio, document, image, structured DB. team จาก Amazon/Meta/MIT/Rivian/Google/Apple/NVIDIA/Samsung AI Center/NASA; advisor Jennifer Chayes (UC Berkeley Dean) + Christian Borgs; early customer GS group (Korean chaebol). thesis: RAG ไม่พอแล้ว ต้อง pre-compute connection structure ทั้งหมดเป็น graph

Pattern หลักประจำรอบ — **"deployment volume" กำลังถูก reality-check ด้วย "reliability constraint"**. ฝั่ง bull: PwC จะ deploy Claude ที่ scale 30K + Advocate Health 167K, Anthropic แยก agent pricing = เตรียม revenue model สำหรับ agent loop หนัก ๆ; ฝั่ง bear: Microsoft paper ชี้ว่า agent loop ที่ลึกเกิน 10 step ใน domain ที่ไม่ใช่ code จะ silently corrupt output; ฝั่ง infrastructure: Graphon เปิดประเด็นว่า RAG ที่เป็น default architecture อาจถึงทางตัน. ผู้ชนะของรอบนี้คือ — บริษัทที่ build agent platform ที่มี **circuit breaker + checkpoint replay + cost observability + multi-model routing** ตั้งแต่ day 1 ไม่ใช่ที่ deploy เร็วที่สุด
