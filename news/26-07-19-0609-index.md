## 🎯 ถ้ามีเวลา 5 นาทีก่อนเข้าออฟฟิศ อ่านอะไรก่อน?

**อันดับ 1 (capability shock):** [01] Kimi K3 — proof point ของ long-horizon agentic ที่แข็งแรงที่สุดที่เคยเห็น; ถ้าทีมกำลังออกแบบ retry/checkpointing/memory compaction สำหรับ agent ที่ต้อง run หลายวัน นี่คือ reference implementation ที่ควรอ่าน. คนที่ยัง price ว่า "closed frontier lead ห่างเป็นปี" ต้องคำนวณใหม่

**อันดับ 2 (infra bet):** [02] Fireworks — 40T token/วัน + 95% specialized เป็นตัวเลขที่บอกทิศทาง cost curve ของ agent workload; ทีมที่ scale agent เกิน 10K call/วัน/ลูกค้า ควรทำ POC swap runtime ให้เสร็จภายในไตรมาส

**อันดับ 3 (distribution reality):** [03] Microsoft Frontier + AWS FDE — ถ้าคุณเป็นทีม startup ที่ทำ orchestration หรือ vertical agent platform, "partnership กับ hyperscaler FDE" คือ distribution ที่โครงการ Fortune 500 จะเข้ามาปีนี้ ไม่ใช่ AWS Marketplace หรือ Azure Marketplace แบบ self-serve

**อันดับ 4 (sovereign hardware):** [04] HPE + NVIDIA Vera — enterprise ในสาย regulated ที่รอ on-prem agent stack ควรเริ่มเขียน RFP รวม Vera CPU + Agent Toolkit เทียบ hyperscaler managed; policy layer (NemoClaw ตาม Q4) จะเป็นจุดที่ vendor lock-in เกิดถ้าไม่คุมตั้งแต่ต้น
