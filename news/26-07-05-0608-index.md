# Daily Brief — 26-07-05 รอบ 06:08 (Bangkok)

## Theme ของรอบ

**48 ชั่วโมงล่าสุด (30 มิ.ย. – 3 ก.ค. 2026) = "implementation คือ moat ใหม่ ไม่ใช่โมเดล"**. หลังจากรอบก่อนที่ pattern เป็น "unit economics ของแต่ละ layer ต้อง prove" (routing, pricing, tooling, security), รอบนี้ pattern shift ขึ้น altitude อีกชั้น — **model layer commoditize แล้ว, สงครามจริงย้ายไปที่ใครทำให้ลูกค้า deploy ได้จริง**

**Microsoft ตั้ง Frontier Company $2.5B + 6,000 engineers** (2 ก.ค.) เป็น signal เด่นสุด — พร้อมมี AWS ประกาศ $1B FDE unit 2 วันก่อนหน้า (30 มิ.ย.) รวมกับ OpenAI Deployment Company $10B และ Anthropic joint venture $1.5B = **frontier lab + hyperscaler 4 เจ้าปล่อย FDE arm ในช่วง 90 วัน รวม $15.1B**. Microsoft ประกาศ Big-4 (Accenture, Capgemini, EY, KPMG, PwC) เป็น channel partner ไม่ใช่คู่แข่ง — signal ว่าตลาดยอมรับแล้วว่า **implementation ต้องเป็น first-party ของ vendor ไม่ใช่ทิ้งให้ SI ทำเอง**

**Salesforce+AWS ปล่อย Agentforce 360 for AWS ขึ้น Marketplace** (2-3 ก.ค.) เป็นสัญญาณคู่ขนาน — Salesforce ยอมเปิดให้ Bedrock อยู่ตรงกลาง, LLM traffic วิ่งใน Trust Boundary (private Hyperforce cloud), CrowdStrike เป็น showcase customer, procurement ผ่าน AWS EDP ตัดขั้นตอน vendor onboarding ปกติ 4-9 เดือน ให้เหลือแค่ commit expansion. Microsoft เดินสูตร **"ส่ง engineer ไปฝังตัว"**; Salesforce+AWS เดินสูตร **"รวมทุกอย่างในสัญญาเดียวให้ CFO อ่านเข้าใจใน 1 หน้า"** — สองสูตรจะแบ่ง Fortune 500 workload กันไปคนละครึ่ง

**Levi Strauss ประกาศ Super Agent บน Microsoft Foundry** — Fortune 500 retailer รายแรกที่ ship orchestration layer ครอบ specialized agent เดิม (ESS, custom SAP, Retail Agent) พนักงาน 15,000 คนเข้าผ่าน Microsoft Teams อย่างเดียว ถามอะไรก็ได้ super-agent route ไป backend ที่เหมาะสม pattern **"specialized-agent first, super-agent later"** ตรงข้ามกับ vendor ที่พยายาม sell "one agent to rule them all" ตั้งแต่ day 1

**Jamf ปล่อย AI Governance for Mac เข้า GA** (1 ก.ค.) — เป็น governance layer ตัวแรกที่ CISO ได้ audit shadow AI ทั้ง fleet ในสัปดาห์เดียว, support Claude Code/Desktop + OpenAI Codex day-1, ใช้ telemetry agent เดิมไม่ต้อง install เพิ่ม. Signal ว่า **shadow AI จะกลายเป็น line item ในงบ CISO ที่ CIO หลบไม่ได้ในครึ่งหลังของ 2026** — endpoint vendor เดิม (CrowdStrike, SentinelOne, Defender) ต้องออก feature เทียบเคียงในไตรมาสถัดไป

Pattern หลักของ 4 ข่าวรวมกัน — **stack agent ของ enterprise 2026 ประกอบด้วย 4 ชั้นที่ชัดแล้ว**: (1) **Implementation layer** — Frontier Co / FDE ที่รับผิดชอบ outcome, (2) **Procurement/runtime layer** — Marketplace + Trust Boundary + Bedrock, (3) **Orchestration layer** — Super Agent + specialized backend, (4) **Governance layer** — endpoint discovery + policy + report. **ครึ่งหลัง 2026 = การประกอบ 4 ชั้นนี้เข้าด้วยกันในองค์กรจริง — ไม่ใช่การเลือก vendor แล้ว, เป็นการวาง architecture ทั้ง stack**

- **Salesforce+AWS Agentforce 360 on Marketplace = procurement เป็นสมรภูมิใหม่** — ตัด vendor onboarding 4-9 เดือน ให้เหลือแค่ EDP commit expansion, LLM traffic อยู่ใน Trust Boundary + Bedrock ที่ audit ง่าย. **Builders**: SaaS ที่ยัง lock customer ใน cloud ตัวเอง+billing ตัวเองต้องคิด margin trade-off ยอมแบ่ง 5-10% ให้ marketplace vs รักษา margin แต่โดน bypass. **Business ในไทย** ที่ใช้ Salesforce+AWS อยู่แล้ว (SCB, KTC, AIS, retail) คุย procurement ทันทีว่า EDP ปัจจุบันครอบตัวนี้ไหม + CISO audit log/audit trail ของ Bedrock region + PDPA cross-border transfer
