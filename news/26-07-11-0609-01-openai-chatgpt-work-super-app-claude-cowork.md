# OpenAI เปิด ChatGPT Work + GPT-5.6 Sol — super app ตอบโต้ Claude Cowork แบบตรง ๆ ปิดฉาก "chatbot era"

## TL;DR
- OpenAI ประกาศ (9 ก.ค.) เปิด **ChatGPT Work** — super app ที่รวม chat + Codex + browser ไว้จุดเดียว, ทำงาน multi-step ที่กินเวลาหลายชั่วโมงแบบ autonomous, powered ด้วย GPT-5.6 Sol โมเดลใหม่ที่บริษัทเคลม "**54% เร็ว/ประหยัดกว่าคู่แข่ง**"
- Roll out วันเดียวกันบน web + mobile — เริ่มที่ **Pro / Enterprise / Edu** ก่อน แล้ว Plus / Business ตามในไม่กี่วัน. ราคา = อยู่ใน plan เดิม, credit ใช้ pool เดียวกับ Codex ($100–200 / dev / เดือน โดยเฉลี่ย)
- **นี่คือ frontal assault ต่อ Claude Cowork** (Anthropic เปิดตัว ม.ค. 2026) — สงคราม "agent super-app" ระหว่าง 2 lab ใหญ่จะกำหนด default ของ knowledge worker 2027 ทั้งอุตสาหกรรม

## เกิดอะไรขึ้น
วันพฤหัสฯ 9 กรกฎาคม 2026 OpenAI เปิดตัว **ChatGPT Work** — สิ่งที่บริษัทเรียกเองว่า "super app" ตัวแรกอย่างเป็นทางการ. ต่างจาก ChatGPT เดิมที่เป็น chat interface กับ tool ที่ต้อง toggle เข้า-ออก, ChatGPT Work รวม chat, Codex (coding agent), และ browser autonomy เข้าเป็น surface เดียว. ผู้ใช้มอบหมายงาน multi-step กินเวลา "หลายชั่วโมง" — สร้างเอกสาร, presentation, website — แล้วเดินออกจากโต๊ะได้เลย. Agent จะไปเปิด browser, run code, กลับมาพร้อมชิ้นงานเมื่อเสร็จ

Engine อยู่ที่ **GPT-5.6 Sol** — โมเดล frontier ตัวใหม่ที่ debut พร้อมกัน. OpenAI เคลมว่า Sol เร็วขึ้น 54% และประหยัดกว่ารุ่น 5.5 ที่ทดแทน. เผยแพร่วันเดียวกับที่ประกาศบน **Microsoft 365 Copilot** ว่า GPT-5.6 กลายเป็น "preferred model" ทันที — สัญญาณว่า Microsoft distribution channel เก่าที่เคยเป็น constraint (ต้องผ่าน Copilot preview cycle) หายไปแล้ว. Sol ยัง **ล่าช้าจากกำหนดเดิม** เพราะ US government ขอเลื่อนช่วง มิ.ย. ด้วยเหตุผล national security

Rollout เริ่มที่ **ChatGPT Pro, Enterprise, Edu** วันแรก แล้ว Plus + Business ต่อในไม่กี่วัน. Pricing รวมอยู่ใน plan เดิม, credit คิดจาก pool เดียวกับ Codex ที่ปัจจุบันเฉลี่ย $100–200 / developer / เดือน. OpenAI เปิดเผยว่า Codex user ใน Business + Enterprise **โตขึ้น 6 เท่าตั้งแต่มกราคม 2026** — พื้นฐานผู้ใช้ที่พร้อมโดยไม่ต้อง onboard ใหม่

ที่สำคัญ ChatGPT Work คือ **response โดยตรง** ต่อ Claude Cowork ของ Anthropic ที่เปิดตัวเดือนมกราคม 2026 — agent ที่ plan + execute multi-step task ได้แบบ autonomous. Anthropic ต่อยอด Cowork ด้วย web + mobile access เมื่อสัปดาห์ที่แล้ว พร้อม double usage limit สำหรับ Max users ถึง 5 ส.ค. — ทั้งสองเจ้ากำลังเร่งขยาย surface area ก่อนงบ Q3 ปิด

## ทำไมสำคัญ
"Chatbot era" กำลังจะจบใน 6–12 เดือนข้างหน้า. Interface ที่กลายเป็น default คือ **agent surface ที่ทำงานให้ตอนคุณไม่อยู่หน้าจอ** — ไม่ใช่ chat window ที่ถามตอบ. ChatGPT Work + Claude Cowork = 2 super app ที่ยึด mind share knowledge worker Fortune 500 ก่อนใครในตลาดนี้

ตัวเลข 54% cheaper/faster ของ Sol น่าเชื่อถือแค่ไหนต้องรอ third-party benchmark — แต่ signal เชิง strategy ชัดเจน: OpenAI **ยอมกิน margin ตัวเอง** เพื่อไม่ให้ราคาเป็นเหตุผลที่ enterprise ย้ายไป Grok 4.5 (xAI ประกาศวันเดียวกันสัปดาห์นี้ ที่ $2 / $6 per million tokens — 60% ถูกกว่า Opus 4.8 + GPT-5.5). Pricing pressure จากด้านล่าง + ด้าน UX จาก Anthropic กดดันให้ OpenAI ต้องเปิด super app ตอนนี้ไม่ใช่ Q4

Pattern สำคัญที่มองข้ามได้ยาก: **Codex user โต 6x ใน 6 เดือน** — coding agent ไม่ใช่ niche developer tool อีกแล้ว, แต่กลายเป็น consumption engine ที่ enterprise ตั้งงบ $100–200 / developer / เดือน ไว้แล้ว. ChatGPT Work ใช้ pool credit เดียวกันหมายความว่า enterprise ที่จ่ายค่า Codex อยู่แล้วเปิด agent ทำงาน multi-step "ฟรี" ในสายตา CFO — barrier adoption แทบเป็นศูนย์

Cross-reference กับข่าว Microsoft Frontier Co $2.5B + AWS FDE $1B สัปดาห์ก่อน — pattern เดียวกัน: **hyperscaler + frontier lab เดิมพัน deployment velocity มากกว่า model quality**. ChatGPT Work คือชิ้นสุดท้ายที่บอกว่าตลาด knowledge work software 2027 = "agent + credit meter" ไม่ใช่ "seat license"

## มุม AI Agent Platform
- **Builders** — startup ที่สร้าง productivity agent (Adept, MultiOn, Rabbit, Reflect) โดน compress runway จากสองด้าน: (1) ChatGPT Work + Claude Cowork ยึด default entry point, (2) enterprise budget ถูกดูดเข้า Codex/Cowork credit pool. Framework builder (LangChain, LlamaIndex, CrewAI) ต้องเน้น **BYO-model + observability + policy control** — เพราะ business ที่ยัง lock-in ไม่ได้กับ OpenAI/Anthropic โดยตรง (regulated, air-gapped, sovereign) คือ TAM ที่เหลือให้แข่ง. Vertical agent (legal, healthcare, finance) ยังปลอดภัยกว่า generalist productivity agent ที่จะโดน super app compete หัวชนกัน

- **Users / business ในไทย** — **ทดสอบ ChatGPT Work ทันทีบน Pro / Enterprise seat ที่มีอยู่** ก่อนตัดสินใจ pilot ใหม่. Task ที่เคยให้ intern ทำ 4 ชม. (research report, first-draft deck, competitive scan) ลอง delegate ให้ agent 20 นาทีดู — ถ้า output ใช้ได้ 70% ก็ประหยัดเวลา leverage สูงมาก. บริษัทที่ยังอยู่บน ChatGPT Team / Plus เก่า ๆ ที่ไม่มี agent access ควรวางแผน upgrade เร็วขึ้น 1 quarter เพราะ productivity gap กับคู่แข่งจะขยับเร็วกว่าที่คาด

- **Ecosystem** — SI ไทย + APAC (Accenture, Deloitte, Bluebik, G-Able, MFEC) จะเจอ RFP รูปแบบใหม่ปลายปี: "**เชื่อม ChatGPT Work / Claude Cowork เข้า SAP / Oracle / internal SharePoint ให้ได้ในสัปดาห์**". Integration + change management กลายเป็นคอขวด, ไม่ใช่ model access. Microsoft 365 Copilot ecosystem โดนกดดันหนักสุด — เพราะ ChatGPT Work เข้ามาแข่งบน desktop ตรง ๆ ในขณะที่ Copilot ยังต้องผูกกับ Office license. GPT-5.6 ที่ landing ใน Copilot วันเดียวกันเป็น hedge ของ Microsoft ที่รักษาแนวรั้ว distribution
