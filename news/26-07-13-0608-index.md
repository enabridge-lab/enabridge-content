# Daily Brief — 26-07-13 รอบ 06:08 (Bangkok)

## Theme ของรอบ

**48 ชั่วโมงล่าสุด (1–12 ก.ค. 2026) = "protocol + economics reboot"**. รอบที่แล้ว (12 ก.ค.) เห็น 4 middleware layer ของ enterprise agent stack ปักธงเรียบร้อย — federal-grade, gateway, CX control, VROI. รอบนี้ pattern **shift ลึกกว่า altitude เดิม**: **layer ล่างสุดของ stack — protocol เอง — กำลัง reboot ครั้งใหญ่**, และ **layer บนสุด — economics ของ SaaS ทั้งอุตสาหกรรม — กำลังโดน redistribute**. ระหว่าง 2 ปลายนี้ **infra layer** เริ่มยืนยันตัวเองว่าใครคุม cost per token = คุม value; **incumbent networking** เริ่มยัด agent gateway ลง license เดิมเพื่อ commoditize pure-play startup

**MCP ปล่อย release candidate spec 2026-07-28** (RC lock 25 พ.ย. 2025) — Anthropic เรียกว่า "the largest revision since launch". Ship 28 ก.ค. พร้อม **stateless core** (session state ออก, tools/list cacheable, route ผ่าน Mcp-Method header), **OAuth 2.1 + OpenID Connect alignment** (token replay ข้าม server ทำไม่ได้ enforce ที่ protocol level), **Enterprise-Managed Authorization** promoted เป็น stable (IdP กลาง control access, RBAC + audit log built-in), และ **extensions framework** แบบ reverse-DNS. 78% ของ enterprise AI team ใช้ MCP ใน production แล้ว, 28% Fortune 500 รัน MCP server เอง, SDK downloads 97M/เดือน — 2026-07-28 คือ threshold ที่ MCP ข้ามจาก dev toy เป็น protocol ที่ CISO เซ็นได้. 12-month deprecation window เป็น gift ให้ vendor pressed to migrate

**Together AI ปิด Series C $800M ที่ $8.3B valuation** (1 ก.ค.) — **Aramco Ventures นำ**, NVIDIA + Vista + General Catalyst + Emergence + March + Pegatron + SentinelOne S Ventures ตาม. ARR ทะลุ **$1B ใน Q2**, open-source model usage บน platform **โต 3x YoY**, plan scale capacity **50x ใน 5 ปี**. Split ระหว่าง commercial fund ให้ frontier lab (OpenAI $10B, Anthropic $50B, Microsoft Frontier $2.5B) กับ picks-and-shovels ให้ neocloud (Together $800M + Cerebras/Groq/SambaNova + Middle East backer) กำลังชัดขึ้น — **workload shift ไป agent = cost per token เป็น value ตัวจริง มากกว่า benchmark score**

**Citrix เปิด NetScaler MCP Gateway** (9 ก.ค.) — control plane เดียวสำหรับ route/auth/monitor/rate-limit agent traffic ไป backend MCP servers, ใช้ single-pass architecture ของ NetScaler เดิม. **Bundled ฟรีในราคา Citrix Platform License / Universal Hybrid Multi-Cloud** — ลูกค้าเก่าเปิดใช้ได้เลยไม่ต้อง sign PO ใหม่. Claude Code private tech preview เปิดคู่ (NetScaler AI Gateway หน้า Claude Code เป็น single control point). Pattern **3 layer ของ agent gateway** โผล่ชัด: hyperscaler-native (Azure/AWS/Google) + incumbent networking (Citrix/F5/Cisco/Palo Alto) + agent-native startup (Arcade/Portkey/Kong AI/Nutanix) — pure-play startup จะโดน commoditize ใน 12-18 เดือนถ้าไม่มี depth ชัด

**Gartner ปักธง agentic arbitrage $234B** (1 ก.ค.) — enterprise application software spend ที่ **"at risk"** ระหว่างวันนี้ถึงปี 2030 คือ **~20% ของ SaaS ทั้งตลาด**. Definition: agent complete task ข้ามหลาย system → user ไม่ต้องเข้า UI ของ SaaS แต่ละตัว → link ระหว่าง user growth กับ revenue growth หลุด. Managing VP George Brocklehurst ใช้คำว่า **"agentic AI changes the economics of software"**. Winner ตาม Gartner คือ AI-native startup + service provider ที่ทำ agentic layer ข้ามระบบ — กินทั้ง existing spend + incremental budget ที่ ROI ปลดล็อก

Pattern หลักของ 4 ข่าวรวมกัน — **stack agent ของ enterprise 2026 กำลัง reset ทั้ง 3 layer พร้อมกัน**: (1) **protocol layer** (MCP 2026-07-28 stateless + OAuth 2.1 + EMA); (2) **infra layer** (Together AI $8.3B ยืนยัน open-source inference thesis); (3) **control layer** (Citrix ยัด MCP Gateway ลง license, incumbent bundling war เริ่ม); (4) **economics layer** (Gartner $234B — SaaS pricing model UI-heavy จะพังจากบน). 4 ข่าวนี้ตอบคำถามเดียวกันของ CIO: **"เตรียมสถาปัตยกรรม + งบ + procurement pipeline อย่างไรให้รอด 3-5 ปีข้างหน้า"** — window การเลือกข้างเปิดอีกไม่นาน

## TL;DR

- **MCP 2026-07-28 = threshold ที่ protocol ผ่าน "CISO gate"** — session state ออก, OAuth 2.1 เข้า, EMA stable, extensions framework เปิด. **Builders** ที่รัน MCP server อยู่ต้องจัดสรร sprint หน้าให้ migration (session externalize + tools/list idempotent + EMA-ready auth); ถ้าใช้ SDK ของ Anthropic/LangChain/LlamaIndex จะ lag น้อยกว่าคน hand-roll. **Business** ที่ evaluate MCP vendor ให้ถามว่า migrate ไป spec ใหม่หรือยัง — ถ้ายัง อย่า sign 2-year procurement. **Ecosystem** ที่ได้ประโยชน์: gateway vendor (Citrix, Nutanix, Kong, Salt) + IdP (Okta, Entra, Ping) — เพิ่ง get protocol-level moat โดยตรง

- **Together AI $8.3B = picks-and-shovels thesis ผ่าน validation รอบใหญ่** — Aramco Ventures นำ + ARR ทะลุ $1B + open-source usage 3x + plan capacity 50x. Money split ชัด: frontier lab commercial fund (OpenAI/Anthropic/Microsoft Frontier) vs infra fund (Together/Cerebras/Groq/SambaNova + Middle East). **Builders** ต้อง hybrid routing — open-source neocloud สำหรับ high-volume routine call, frontier API สำหรับ complex reasoning; savings 5-10x. **Business** ที่ใช้ vendor agent platform ต้องถามว่ามี open-source routing option ไหม — ถ้าไม่มี ผู้ใช้ (คุณ) จะรับต้นทุนตอน closed API ขึ้นราคา. **Ecosystem**: Gulf capital pool (Aramco, G42, MGX) เข้ามาที่ AI infra จริงจัง — neocloud footprint กระจาย Gulf/India/SEA จะกลายเป็น distribution advantage ใน compliance-sensitive market

- **Citrix NetScaler MCP Gateway = incumbent bundling war เริ่มแล้ว** — pure-play agent gateway startup จะเจอตัวเดียวกับที่ pure-play SDN เจอตอน VMware NSX bundling, ที่ SDN startup เจอตอน Cisco ACI. **Builders** ที่ทำ gateway product ต้อง differentiate เชิง depth (agent-specific policy DSL, prompt-level guardrails, cost attribution ต่อ agent/team, curated MCP marketplace); ถ้า agent app ให้ default assume traffic วิ่งผ่าน enterprise gateway. **Business** ที่จ่าย Citrix License อยู่แล้ว ต้องตรวจว่า MCP Gateway มาให้ฟรีก่อนซื้อ startup ตัวใหม่ — ปีนี้จะมี hidden bundling หลายเจ้า. **Ecosystem** gateway startup รอบใหม่ต้อง (1) integrate กับ NetScaler/F5/Cloudflare เป็น complement, หรือ (2) ยิงไป vertical/regulated (healthcare, finance, gov) ที่ incumbent ยังตามไม่ทัน

- **Gartner $234B agentic arbitrage = CIO wake-up call ที่จะเข้า board deck ทุกที่** — 20% ของ SaaS ทั้งตลาดจะไหลผ่าน agent layer ภายในปี 2030. SaaS pricing model ที่คิดต่อ seat/session/UI usage จะ decouple จาก value. **Builders** ต้อง shift pricing ไป outcome-based (ต่อ ticket resolved, ต่อ document processed, ต่อ deal closed); instrumentation ที่วัด outcome ได้จริง = moat. **Business** ใช้ report นี้เป็น mandate ครึ่งหลังของปี — audit SaaS ทุกตัวว่า "agentic layer" คืออะไร, มี MCP endpoint ไหม, ต้อง pilot outcome-priced contract ก่อน renewal. **Ecosystem** ที่ได้ role ใหญ่ = SI (Accenture, Deloitte, EY, TCS, Infosys) ที่ position เป็น "redesign workflow around agent" — คนที่ยัง "back-end system + no agent story" จะเจอราคาถูก compress ใน 3-4 ปี

- **Pattern หลักของ 4 ข่าวรวมกัน — enterprise agent stack 2026 กำลัง reset ทั้ง 3 layer พร้อมกัน**: (1) **protocol layer** — MCP 2026-07-28 stateless + OAuth 2.1 + EMA เป็น stable; (2) **infra layer** — Together AI $8.3B ยืนยัน open-source inference thesis, cost per token = value; (3) **control layer** — Citrix ยัด MCP Gateway ลง license เดิม, incumbent bundling war เริ่ม; (4) **economics layer** — Gartner $234B — SaaS pricing model UI-heavy จะพังจากบน. 4 ข่าวนี้ตอบคำถามเดียวกันของ CIO: **"เตรียมสถาปัตยกรรม + งบ + procurement pipeline อย่างไรให้รอด 3-5 ปีข้างหน้า"**

- Action ตรงสำหรับ **AI Agent Platform builders**:
  - (1) **Migrate MCP server ไป 2026-07-28** ทันที — externalize session state, idempotent tools/list, EMA-ready auth; SDK vendor beta จะมาก่อน 28 ก.ค.
  - (2) **Hybrid routing เป็น default** — open-source neocloud (Together/Fireworks/Groq) สำหรับ routine call + frontier API สำหรับ reasoning step; instrument cost per token per agent
  - (3) **Design for enterprise gateway pass-through** — assume traffic วิ่งผ่าน Citrix/F5/Cloudflare/Nutanix; retry logic + telemetry format + auth flow ให้เข้ากับ gateway standard
  - (4) **Outcome-based pricing + instrumentation** — ตอบคำถาม CFO ให้ได้ว่า agent ของคุณ deliver outcome เท่าไร ต่อ ticket/document/deal; ถ้ายังคิดต่อ seat จะแพ้ Gartner filter รอบ renewal

- Action ตรงสำหรับ **Business / users** ที่ deploy AI Agent:
  - (1) **MCP audit ก่อน renewal** — ถามทุก MCP vendor ว่า migrate ไป 2026-07-28 spec ตอนไหน; ถ้าไม่ตอบ = อย่า sign 2-year
  - (2) **Cost stack review** — ตรวจว่า agent stack ปัจจุบันมี hybrid routing ไหม; ถ้าไม่มี → ตั้ง budget รับ closed API price shock ปลายปี
  - (3) **License audit ของ incumbent networking** — Citrix Platform License มี MCP Gateway ฟรี, F5/Cisco/Palo Alto อาจ follow ในไตรมาสถัดไป — ตรวจก่อนซื้อ startup ใหม่
  - (4) **Board deck ครึ่งหลังของปี** — Gartner $234B agentic arbitrage ต้องอยู่ใน slide; ใช้เป็น mandate ให้ audit vendor SaaS ทุกตัวว่ามี agent story ที่วัด outcome ได้
