# Sierra ระดมทุน $950M ที่ valuation $15B — Bret Taylor พิสูจน์ "AI agent vertical SaaS" คือ category จริง $150M ARR ใน 7 ไตรมาส

## TL;DR
- 4 พ.ค. 2026 Sierra Technologies ของ Bret Taylor ปิด Series E **$950M led by Tiger Global + GV** (Benchmark/Sequoia/Greenoaks ร่วม) — valuation **$15.8B** จาก $10B เมื่อ 8 เดือนก่อน

## เกิดอะไรขึ้น

วันที่ 4 พ.ค. 2026 **Sierra Technologies** บริษัท AI customer agent ของ Bret Taylor (อดีต co-CEO Salesforce + chair OpenAI) กับ Clay Bavor (อดีต Google) ปิด Series E **$950M** lead โดย Tiger Global + GV (Alphabet's venture arm) Benchmark, Sequoia, Greenoaks ร่วม ทำให้ valuation pre-money $15B (post-money บางรายงาน $15.8B) — โต 50% จาก $10B ที่ปิด round $350M เมื่อ 8 เดือนก่อน

ตัวเลข business ที่ Sierra เปิดออกมา (ตอน Sacra เปิด report + บริษัท comment กับสื่อ): **ARR $150M ในเดือน ม.ค. 2026** จาก $130M ปลายปี 2025 และ $26M ปลายปี 2024 — เพิ่ม 5.7x ภายในปี และโตจาก $100M (พ.ย. 2025) เป็น $150M ใน 2 เดือน รายงาน "Q3 2025 ปิด first $50M quarter" + เข้าปี 3 ที่ $150M ARR หลัง launch ก.พ. 2024 = 7 ไตรมาส 0-to-$150M (Snowflake, Wiz, OpenAI ARR เคยทำได้ Sierra เพิ่มชื่อตัวเองในรายชื่อนี้)

ลูกค้า: **40%+ ของ Fortune 50** + **1 ใน 3 bank ใหญ่ที่สุดของโลก** + Prudential, Cigna, Blue Cross Blue Shield, Rocket Mortgage Sierra เริ่มที่ customer support บนข้อความ — แต่ตั้งแต่ต.ค. 2025 **voice agent ใช้ traffic แซง text** และ run pace หลายร้อยล้าน AI call ปี 2025 ที่ critical คือ expand ออกจาก support ไปสู่ **insurance claims processing / mortgage processing / healthcare billing** — workflow ที่มี dollar attached ต่อ decision และ regulator มองตา

Round นี้เป็นการ rerate ทั้ง category — ตัวเลขเฉลี่ยของ agentic AI startup ที่ปิด round Q4 2025 ถึง H1 2026 อยู่ที่ ~$155M (ตัวเลข tracker ใน aifundingtracker + new market pitch); Sierra round เกิน 6x ของ category average — และเป็น 6x ของ next biggest agentic round ในไตรมาสเดียวกัน

## ทำไมสำคัญ

Sierra เป็น first proof point ที่ **"agent vertical SaaS" เป็น category จริงไม่ใช่ wrapper ของ frontier model** $150M ARR ใน 7 ไตรมาสคือ aggressive กว่า Snowflake / OpenAI / Anyscale และทำตอนตลาดที่ทุกคนกลัวว่าจะถูก commoditize โดย OpenAI Frontier Workspace + Anthropic Managed Agents บทเรียน: layer ที่อยู่เหนือ frontier model + ลึกเข้าไปใน workflow ของ Fortune 500 — ที่ปฏิบัติการขายผ่าน FDE-style implementation + UX สำเร็จรูป + integrate ลึกเข้าระบบลูกค้า — มี pricing power เทียบเท่า enterprise SaaS แบบ Salesforce/ServiceNow

แต่ valuation $15B ที่ ARR $150M = 100x ARR เป็น multiple ที่ aggressive แม้ในตลาด AI Snowflake IPO 2020 อยู่ที่ ~70x ARR; Wiz private valuation peak ~50x ARR Sierra ที่ 100x = ตลาดเดิมพันว่า ARR จะโตอย่างน้อย 3-4x ในปี + take rate margin ใน vertical workflow จะ stay 80%+ ความเสี่ยงที่ underrated: ถ้า OpenAI หรือ Anthropic ปล่อย **out-of-the-box vertical agent** ที่ใกล้เคียง Sierra — อย่าง 10 banking agent ของ Anthropic ที่เปิด 5 พ.ค. — Sierra ต้อง defend ทุก vertical ด้วย integration depth + customer relationship ไม่ใช่ feature

อีก signal สำคัญ: GV (Google's venture arm) เป็น co-lead — บอก signal ตรงว่า Google มอง Sierra เป็นพันธมิตรที่ Google ไม่ต้อง compete ด้วยตรง ๆ (Gemini ขาย model + agent platform; Sierra ขาย vertical workflow); Microsoft Azure + Anthropic / OpenAI relationship ก็ pattern เดียวกัน — frontier vendor accept ว่า application-layer vertical SaaS เป็น customer ที่ขายโมเดลให้ ไม่ใช่ competitor การที่ Tiger lead = bet ว่า public market exit หรือ secondary sale จะเป็น path ภายใน 18-24 เดือน Sierra เป็น candidate IPO ปี 2027 ที่ชัดเจนที่สุดในรุ่น agent

## มุม OpenBridge

Sierra เป็น playbook ที่ OpenBridge ควรอ่านบรรทัดต่อบรรทัด — แต่ position OpenBridge **ไม่ใช่ alternative ของ Sierra** OpenBridge อยู่ที่ infrastructure layer ที่ Sierra (และ competitor ทุกตัว) ใช้ Sierra build customer agent — แต่ทุก agent ต้องเข้า ServiceNow, Salesforce, Workday, SAP, Workiva, internal API ของลูกค้า ส่วนที่ Sierra ขาย vertical depth + UX OpenBridge ขาย **horizontal connectivity ที่ทุก agent vertical SaaS ใช้** ไม่ว่า Sierra จะ scale ขนาดไหน หรือ Salesforce/ServiceNow ออก agent เอง

Insight ที่ตรงสุด: **agent vertical SaaS เป็น customer segment ใหม่ของ OpenBridge** — ไม่ใช่ ISV เดิมที่ขาย CRM/HR/finance แต่เป็นบริษัทเหมือน Sierra, Decagon, Cresta, Parloa ที่ build agent ของตัวเอง customer พวกนี้ต้อง ingest event + push action ไป SaaS ของลูกค้านับสิบตัว — และเขารู้ว่า build connector เองไม่ scale OpenBridge ควรมี **"powering AI agent platforms" sales motion** แยกจาก enterprise direct: pricing per-event + per-action + revenue share จาก usage ของ Sierra เก็บกับลูกค้าปลายทาง

อีกประเด็น: Sierra expand จาก customer support ไป insurance claims / mortgage / healthcare billing — บอกตรงว่า **next wave ของ agent vertical คือ workflow ที่มี dollar attached + regulator มองตา** OpenBridge ควรลงทุน connector + audit trail สำหรับ insurance core (Guidewire, Duck Creek), mortgage origination (Encompass, Empower), healthcare billing (Epic, Athena, Cerner) ก่อน competitor — connector enterprise ปกติมีอยู่แล้วใน Workato / Tray แต่ที่ "agent-ready" + "audit-grade" + "outcomes-emit" ยังไม่มีใครทำครบ
