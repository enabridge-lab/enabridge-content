# Moment ระดม $78M ที่ AUM ลูกค้า $10 ล้านล้าน — Edward Jones + LPL + Hightower เปลี่ยน wealth management เป็น AI OS

## TL;DR
- 19 พ.ค. Moment (NYC, ก่อตั้งโดยอดีต quant Citadel) ปิด Series C **$78M** นำโดย Index Ventures + a16z + Avra — รวมระดมแล้ว **$134M**
- ลูกค้า standardize บน platform คือ Edward Jones, LPL Financial, Hightower Advisors — รวม AUM **> $10 ล้านล้าน** (จาก $300B เมื่อ 18 เดือนก่อน = **33x growth**)
- 3 agent core: **portfolio construction** (สร้าง custom portfolio จาก natural language ใน "วินาที"), **surveillance** (scan account หา tax/risk/transition opportunity), **compliance** (real-time scan account/transaction/order vs custom rule set)

## เกิดอะไรขึ้น

19 พฤษภาคม 2026 Moment — สตาร์ตอัพที่ Bill Capuzzi กับทีมอดีต quant จาก Citadel Securities ตั้งขึ้นใน NYC — ประกาศปิด Series C **$78 ล้านดอลลาร์** นำโดย **Index Ventures** พร้อม Andreessen Horowitz, Avra, และนักลงทุนเดิม. รวม funding ทั้งหมด $134M. ราวด์นี้ตามมาเร็วผิดปกติ — รอบก่อนยังไม่ครบปี และตัวเลขที่ทำให้ Index Ventures กล้าลงคือ **AUM ของลูกค้าบน platform แตะ $10 ล้านล้าน — จาก $300 พันล้านเมื่อ 18 เดือนก่อน** = growth 33x ที่ระดับ asset ไม่ใช่ revenue

ลูกค้าหลักที่ Moment เปิดเผยพร้อมราวด์ — **Edward Jones** (50,000+ advisor, AUM $2T), **LPL Financial** (ใหญ่ที่สุดของ independent broker-dealer ใน US, AUM $1.7T), **Hightower Advisors** (RIA ระดับ top 10, AUM $200B+). ทั้ง 3 รายเลือก standardize ทั้ง stack ของ trading + portfolio management + compliance ลงบน Moment — แทนที่จะใช้ Charles River, BlackRock Aladdin, หรือ FactSet ที่เป็น incumbent ของวงการ

Product ที่ทำให้ adoption เร็วผิดปกติคือ **3 agent core ที่ทำงานบน asset class + currency เดียวกัน**: (1) **Portfolio construction agent** — รับ instruction เป็น natural language เช่น "build tax-loss harvest portfolio for client age 55, IRA, target 60/40 with municipal tilt" แล้ว generate portfolio ใน "วินาที" + เสนอ trade list พร้อม execute, (2) **Surveillance agent** — scan ทุก account วันต่อวันหา tax-loss opportunity, risk concentration, transition opportunity (เช่น client ใกล้เกษียณ), แจ้ง advisor ก่อน client หาเอง, (3) **Compliance agent** — รัน real-time check ทุก transaction + order vs custom rule set ของแต่ละ firm (Reg BI, fiduciary, internal policy)

จุดที่นักลงทุนเห็น value คือ — Moment **กิน workflow ทั้ง front-to-back office** แทนที่จะขายเป็น tool แยก. Edward Jones ที่มี 50,000 advisor ใช้ Moment เท่ากับเปลี่ยน operating model ทั้งบริษัท ไม่ใช่ pilot ใน 50 office. ARR ที่ implied จาก deal size ของ Edward Jones + LPL + Hightower น่าจะ > $100M แล้ว (จาก press signal "standardize on platform" + ขนาด firm) — เป็น scale ที่ทำให้ valuation Series C นี้ครอบคลุมรวดเร็ว

## ทำไมสำคัญ

นี่คือ proof point ใหญ่ที่สุดของรอบว่า **agentic AI ใน wealth management พ้น experimentation แล้ว — เข้า production replacement ของ incumbent stack**. ก่อนหน้านี้ market เห็น use case ใน robo-advisor (Betterment, Wealthfront) ที่จับ retail แต่ไม่กระทบ infrastructure ของ advisor firm. Moment ทำตรงข้าม — ขาย OS ให้ advisor firm ที่จัดการ AUM trillion-dollar. ลูกค้าระดับ Edward Jones ยอมเปลี่ยนเพราะ pain เดิม (Charles River integration, manual compliance, 4-hour portfolio construction) แก้ไม่ตก ขณะที่ Moment ทำเสร็จใน "วินาที"

มอง 12-18 เดือนข้างหน้า — pattern ที่ Moment ทำซ้ำได้ใน insurance underwriting, claims processing, lending decision, treasury management. ถ้า AUM growth 33x ใน 18 เดือน holds — Moment จะ trading ที่ valuation **$1.5-2B** ภายในปลายปี และ rival ที่กำลังตามมาคือ BlackRock Aladdin ที่กำลังเร่ง AI-native rebuild + FIS ที่ตอนนี้ทำกับ Anthropic ใน financial crimes agent. Wall Street infrastructure incumbents มี 12-18 เดือนจะ ship agent layer ของตัวเอง — หรือยอมโดน Moment-style startup กิน vertical

อีก signal สำคัญ — **enterprise SaaS metric ใหม่ที่นักลงทุนเริ่มใช้คือ "AUM/ARR ratio" + "asset velocity"**. ก่อนยุค AI วัด ARR + NRR + logo count. ตอนนี้ Index Ventures เห็น $10T AUM under management ของ platform = signal ว่า platform "stick" — advisor ที่ใช้แล้วไม่ออก เพราะ workflow ทั้งหมด rewire ใหม่. นี่คือ moat ที่ deeper กว่า typical SaaS contract เพราะ asset ที่ platform บริหารคือ accountability ของ fiduciary advisor — ย้ายระบบ = audit nightmare

## มุม OpenBridge

ไม่ direct เกี่ยว — Moment ขาย vertical OS ไม่ใช่ horizontal integration. แต่มี **3 adjacent insight สำคัญ**:

(1) **Pattern incumbent + AI-native vertical = win playbook** ที่ทำซ้ำได้ใน SEA wealth/insurance. Moment + Edward Jones = AI-native ครอบ legacy firm. OpenBridge สามารถ position ตัวเองเป็น integration backbone ให้ AI-native vertical Thai/SEA startup ที่กำลัง build "Moment for Thai bancassurance" หรือ "Moment for SEA family office" — ที่ต้องต่อ KBank private banking + KTC investment + Bualuang Securities + AIA + Allianz. AI-native startup ไม่อยากเสีย 6 เดือนทำ connector — OpenBridge ขาย connector kit + audit log + compliance map ให้พร้อม

(2) **AUM/asset-velocity เป็น new VC metric** ที่กระทบ pitch ของทุก B2B AI startup. OpenBridge ที่ขาย integration ควรวัด + report metric ใหม่ทำนองนี้: **"connector volume"** (transaction routed/mo), **"system count under integration"** (endpoint connected per customer), **"data velocity"** (events processed/sec). ถ้า scale ไปถึงระดับ billion event/mo + 50+ system/customer — pitch จะ resonate กับ Series A/B investor มากกว่า ARR ปกติ

(3) **Compliance agent คือ entry point ที่ regulated industry ยอม sign แรก** — Edward Jones ใช้ Moment เริ่มจาก compliance agent ก่อน portfolio agent เพราะ pain ที่หนักที่สุดคือ Reg BI/fiduciary check. OpenBridge ที่กำลังขายเข้า Thai bank อาจ position เริ่มจาก **"compliance integration layer"** — ต่อ AML/KYC/STR reporting + ตรวจ outbound transaction vs ป.ป.ง. list + automate IR submission ให้ BOT — ก่อนค่อยขยายเป็น horizontal integration layer ทีหลัง
