# Daily Brief — 26-05-08 รอบ 08:59 (Bangkok)

## Theme ของรอบ

**5–7 พ.ค. 2026 คือสัปดาห์ที่ตลาดเริ่มวัด "ความเร็วของ AI diffusion" ด้วยเครื่องมือใหม่ — แล้วเริ่มขาย "เครื่องมือปิด gap" เป็นแพ็กเกจ** วันที่ 7 พ.ค. OpenAI เปิด **B2B Signals** เป็นครั้งแรกที่ vendor AI ปล่อย diffusion metric รายไตรมาสแบบ Bureau of Labor Statistics — frontier firm ที่ percentile 95 ใช้ AI หนักกว่าค่าเฉลี่ย **3.5 เท่า** (เพิ่มจาก 2.0 เท่าใน เม.ย. 2025 = ช่องว่างขยาย 75% ใน 12 เดือน); message volume อธิบายแค่ 36% ของ gap — อีก 64% มาจาก "ใช้ลึกกว่า" — ส่ง context ยาวกว่า, delegate งาน multi-step (Codex 16 เท่า, ChatGPT Agent, Deep Research), generate output substantive

วันที่ 5 พ.ค. ที่ ServiceNow Knowledge 2026 NVIDIA + ServiceNow เปิด **Project Arc** — autonomous desktop agent ที่ "thinks, writes code, executes, adapts" บน laptop พนักงานโดยไม่ต้อง pre-build workflow; ทำงานในกล่อง **NVIDIA OpenShell** ที่บังคับ policy gate ของ **ServiceNow AI Control Tower** ทุก action; grounded ใน **CMDB + Action Fabric** — agent รู้ enterprise context ตั้งแต่ first run, ไม่ใช่ general-purpose computer-use agent ที่หลงใน workflow; first credible **endpoint-native autonomous agent ที่ ship พร้อม governance** ตั้งแต่ day 1

วันที่ 6 พ.ค. ServiceNow + Accenture เปิด **Forward Deployed Engineering program** — ServiceNow AI-native FDE + Accenture industry FDE ลงสนามในลูกค้าก่อนสร้าง production workflow; เปิด catalog **300+ pre-built AI agent skill + agentic workflow** ที่ deploy ใน 4-8 สัปดาห์ (เดิม 6-12 เดือน); **"deliver value in production ก่อน enterprise rollout"** เป็นการแปลง FDE methodology ของ Palantir + OpenAI Frontier เป็น **product motion ที่ scale ได้** — แก้โจทย์ pilot purgatory ที่ 86% ของ enterprise agent หยุดที่ pilot

วันเดียวกัน IBM Think 2026 เปิด **IBM Enterprise Advantage** — first-of-its-kind **asset-based consulting** ที่ "ขาย asset แทนชั่วโมงคน"; **Context Studio** (GA) ground agent บน schema + process ของบริษัทเอง (accuracy 23% → 78%); **Process Studio** (coming soon) — pilot จริง analyze **1,400 SOP**, ค้นพบ **1,000+ improvement opportunity**, project ลด operating cost **25%+ ใน 18 เดือน**; IBM-SAP ผูก **A2A interoperability** — IBM Consulting Advantage agent manage SAP Joule agent ได้ทันที — first concrete cross-vendor agent orchestration ระดับ deployed

Pattern ใหญ่ที่เห็น — ตลาดยอมรับสามอย่าง: (1) **AI diffusion = competitive metric** ที่ board ทุกบริษัทจะ track ใน Q3 (OpenAI ตี baseline แล้ว); (2) **ROI ของ AI = engineering + organizational problem** ไม่ใช่ license sale — 64% ของ frontier gap คือ depth + delegation + governance ไม่ใช่ message count; (3) **consulting แปลงเป็น product** — Accenture+ServiceNow productize FDE, IBM productize SOP-to-agent extraction, Anthropic+Blackstone build services-firm-as-product — ใครยังขายชั่วโมงคนแบบเดิมจะถูกบีบเข้า commodity zone ใน 24 เดือน

## TL;DR สำหรับ Yoh

- **AI diffusion เริ่มมี benchmark public** — OpenAI B2B Signals วัด "intelligence per worker" รายไตรมาส; frontier firm = 3.5x ของ median (เพิ่มจาก 2.0x ใน 12 เดือน = ช่องว่างขยาย 75%); Codex 16x; pattern เดียวกับ NPS ปี 2003 ที่กลายเป็น KPI มาตรฐานใน 5 ปี — board ของทุก Fortune 500 จะถาม "เราอยู่ percentile ไหน" ใน Q3; ลูกค้าไทย Fortune 100 อยู่ percentile 30-50 — มี runway 12-18 เดือนปิด gap
- **ROI = engineering + organizational, ไม่ใช่ license sale** — 64% ของ frontier gap คือ depth + delegation + governance; Microsoft Work Trend Index (5 พ.ค.) ก็ชี้ว่า 67% ของ AI impact = organizational factor; รวมกับ IBM Process Studio claim 25% cost reduction จาก SOP redesign — ตลาดยอมรับว่า "rollout license = pilot purgatory; rollout ที่ workflow redesign + governance + enablement = compound advantage" ขายชั่วโมง consultant แบบเดิมตายในรุ่นนี้
- **Endpoint-native autonomous agent + governance day 1** — Project Arc ทำให้ desktop ของพนักงานกลายเป็น governed agent runtime ผ่าน NVIDIA OpenShell sandbox + ServiceNow AI Control Tower; ใน Q3 CISO+CIO จะไม่ผ่าน agent ที่ไม่มี OpenShell-equivalent; NVIDIA ขยับจาก "ขาย GPU" → "ขาย agent runtime" — margin จะย้ายจาก hardware (~70%) ไป software/runtime (~85%+) เกม Microsoft Agent 365 vs ServiceNow ที่ "ใครเป็น governance layer ของ desktop agent บริษัท" จะตัดสินใน Q4
- **Consulting industry แปลงเป็น product** — Accenture+ServiceNow FDE program 300+ skill, IBM Enterprise Advantage asset-based, Anthropic+Blackstone services-firm-as-product (4 พ.ค.); pricing model จะเป็น **outcome-based** ไม่ใช่ T&M; Accenture เป็น partner ทั้ง OpenAI + Anthropic + ServiceNow — วาง "ครึ่งทุกฝั่ง"; Big 4 ที่ไม่ pivot จะถูก commoditize ภายใน 24 เดือน
- **A2A protocol บูม** — IBM-SAP A2A interop = first concrete cross-vendor agent orchestration in production; 7 เจ้าใหญ่ commit แล้ว (Microsoft, AWS, Salesforce, ServiceNow, SAP, IBM, Google); OpenAI ใช้ MCP เป็นหลัก ยังไม่ explicit commit A2A — Q4 จะตัดสินว่า MCP+A2A converge หรือแตกเป็นสองค่าย
