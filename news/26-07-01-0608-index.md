# Daily Brief — 26-07-01 รอบ 06:08 (Bangkok)

## Theme ของรอบ

**24 ชั่วโมงล่าสุด (30 มิ.ย. – 1 ก.ค. 2026) = "agent distribution + delivery layer กำลังกลายเป็น moat ใหม่"**. หลังจาก dual-track regulatory choreography เมื่อรอบที่แล้ว (OpenAI Sol/Terra/Luna + government gate), รอบนี้ pattern ที่เห็นชัดคือ frontier labs เร่งสร้าง **distribution gravity** ผ่าน SI firm + ML platform + protocol layer — ไม่ใช่ผ่าน direct sales แล้ว

**Globant × Anthropic** (30 มิ.ย.) เป็น signal เด่นที่สุดของรอบ — Globant เข้าเป็น **Preferred Services Partner** ใน Claude Partner Network, ปลด 28,500 Globers ทั่วโลกเข้า Claude + certification track. AI Pods (หน่วยบริการ agent-orchestrated) ตอนนี้ลูกค้า top 20 ของ Globant ใช้ไปแล้ว 40% — partnership ทำให้ทุก Pod default Claude สำหรับ vertical media/airline/gaming/hospitality. รวมกับ Microsoft Foundry GA เมื่อ 28 มิ.ย. + SpaceX compute deal 29 มิ.ย. = Anthropic build distribution layer ครบทุก layer ใน 3 วัน — cloud + advisor + compute. Playbook คล้าย Red Hat + VMware + Snowflake — winner คือคนที่มี distribution gravity ไม่ใช่ best model

**CoreWeave ARIA** (29 มิ.ย.) เปิด public preview พร้อม W&B Weave entering **General Availability** ในวันเดียวกัน — first material proof ว่าดีล **$1.4B Weights & Biases** ที่ปิดเมื่อ พ.ค. 2025 ไม่ใช่แค่ data play แต่เป็น **agent platform play**. ARIA เป็น AI research agent ที่อ่าน experiment data ของทีม ML — leverage data moat ที่ unique: ~1 พันล้าน experiment runs + trillions of metrics จาก W&B platform. CoreWeave ขยับจาก GPU rental shop → full-stack AI platform with agent layer. Pattern: GPU cloud → agent platform via ML platform M&A เป็น playbook ที่ Nvidia, Crusoe, Lambda จะตามมา

**Taktile $110M Series C** (24 มิ.ย., **lead by Goldman Sachs Alternatives**) เป็น signal ใหญ่ว่า **regulated financial services กลายเป็น vertical ที่ agentic AI mature เร็วสุด**. Taktile = modular Agentic Decision Platform ที่ผสม AI agent + business rule + human oversight สำหรับ underwriting, claims processing, AML detection. ลูกค้า includes Coinbase, Klarna, Mercury, Zilch. Goldman ลงเป็น lead = signal ว่า bank/insurer ของ Goldman เริ่มถามว่าควรซื้อ Taktile + Goldman เองอาจเป็น customer. รวมเงินระดมตลอด $184M

**Cognizant × ServiceNow via MCP** (18 มิ.ย.) คือ proof point ใหญ่ที่สุดถึงตอนนี้ของ **MCP ในระดับ enterprise vendor-to-vendor**. Cognizant Neuro AI Multi-Agent Accelerator (**open source บน GitHub**) ตอนนี้ orchestrate ServiceNow AI Agents ผ่าน MCP โดยตรง — ไม่ต้องเขียน custom connector. รองรับ multi-model (Claude, GPT, Gemini, Llama) + multi-cloud. ก่อนหน้านี้ MCP use case ส่วนใหญ่เป็น LLM-to-tool — ตอนนี้ ServiceNow (ITSM #1 ใน Fortune 500) เปิด agent ผ่าน MCP standard = MCP กลายเป็น **enterprise integration protocol จริง ๆ**, Linux ของ agent interop

Pattern หลักของ 4 ข่าวรวมกัน — **moat ใหม่ของ AI labs ไม่ใช่ model quality แต่เป็น "ใครเป็นคนกด deploy"**. Anthropic เซ็น SI firm (Globant, PwC). CoreWeave acquire ML platform แล้ว build agent บนนั้น. Goldman ลงเงินใน vertical agent ตรงที่ regulator/budget เอื้อ. Cognizant + open-source orchestrator + open protocol = neutral layer ที่ enterprise multi-vendor ยอม. ทั้งหมดเป็น distribution play ไม่ใช่ model play — เพราะ model commoditize เร็วกว่าที่คาด

## TL;DR

- **Globant × Anthropic alliance คือ proof ว่า Anthropic เลือก distribution moat แทน model moat** — เซ็น PwC 30K certified พ.ค., Globant 28.5K มิ.ย., รวม certified Claude deployer จะแตะ 100K ปลายปี เทียบ Salesforce trailblazer 18 ปีกว่าจะแตะ million. AI Pods (40% top 20 accounts adopt แล้ว) คือ **human-agent team unit** ที่ Globant ขายเป็น service line — ขายเร็วกว่า product license. **Agent platform builders** ต้อง assume SI-firm partnership เป็น GTM default — ถ้าไม่มี certified consultant 1K+ คน deploy Fortune 1000 ยาก. **Business** ใน Thai bank/hospitality/airline — เช็คก่อนว่า SI partner ใน region มี Claude certified team หรือไม่ ถ้าไม่มี ต้องเตรียม budget upskill ในบ้าน

- **CoreWeave ARIA + Weave GA = pure infrastructure player ก้าวขึ้น agent platform layer สำเร็จ** — ที่ AWS Bedrock, Google Vertex Agent Builder ยังมี friction. ARIA ชนะเพราะ specialize ใน ML/AI engineering workflow + มีลูกค้า built-in 1,400 team + data unique (1B runs, trillions metrics) ที่ third-party เข้าไม่ถึง. **Agent tooling builders** (LangChain, Langfuse, Helicone, Braintrust, Arize) ต้อง shift จาก "framework ดีสุด" → "integration ลึกกับ data + deploy workflow ปลายทาง" หรือ vertical specialization. **ML/AI team ในไทย** ที่ใช้ W&B อยู่แล้ว (CP, SCB, KBank Tech, Bualuang) ลองเข้า preview waitlist ทันที — ARIA ลด research overhead 30-50% ตาม early user report

- **Taktile $110M (Goldman lead) คือ confirmation ว่า regulated financial services = vertical ที่ agentic AI mature เร็วสุด** — workflow ชัด, ROI วัดได้, budget 6-7 หลัก, regulator demand audit trail ที่ Taktile design ตอบโจทย์. Architectural philosophy ที่ชนะ = **decision-tree-first + rule-engine-first** แล้วใส่ LLM/agent ใน step ที่ unstructured เท่านั้น — error mode predictable, audit log ครบ. **Agent framework builders** (LangGraph, OpenAI Agents SDK, Dust, Cognigy, AutoGen) ต้องทำ "policy-as-code + agent + human handoff" เป็น first-class primitive. **Thai banking/insurance** (KBank, SCB, AIA, Bangkok Insurance) — buy เร็วกว่า build 6-12 เดือน แต่ Thai regulator (BoT, SEC, OIC) ยังไม่มี agentic framework — first mover ต้อง engage regulator + governance paper

- **Cognizant × ServiceNow via MCP คือ MCP graduating จาก LLM-to-tool → enterprise vendor-to-vendor protocol** — ServiceNow (ITSM #1 Fortune 500) + Anthropic + OpenAI + Google + Cognizant = critical mass ที่ทำให้ MCP เป็น **Linux ของ agent interop**. Cognizant position เป็น **neutral orchestrator** (open source + open protocol + multi-vendor) — strategic opposite ของ Salesforce/Microsoft. **Agent platform builders** ต้อง assume MCP เป็น must-have integration primitive ตั้งแต่ Q3 — ถ้ายัง require custom SDK สำหรับ ServiceNow/Salesforce/SAP/Workday เสีย deal. **Thai enterprise** (PTT, SCG, AIS, True, CP) ที่ใช้ ServiceNow + SAP + Salesforce — MCP-based orchestration เป็น path ที่ governance ทีมยอมง่ายกว่า "agent platform ใหม่จาก single vendor"

- **Pattern หลักของ 4 ข่าวรวมกัน — moat ใหม่ของ AI labs ในครึ่งหลัง 2026 ไม่ใช่ model quality แต่เป็น "ใครเป็นคนกด deploy"**. Anthropic เซ็น SI firm. CoreWeave acquire ML platform แล้ว build agent บนนั้น. Goldman ลงเงินใน vertical agent ตรงที่ budget/regulation เอื้อ. Cognizant + open-source + open protocol = neutral layer ที่ multi-vendor enterprise ยอม. ทั้งหมดเป็น distribution play ไม่ใช่ model play — เพราะ model commoditize เร็วกว่าที่คาดในไตรมาสที่ผ่านมา (GPT-5.6 split 3 tier, Claude เข้า Foundry, Qualcomm-Modular ปลด CUDA, MCP stateless)

- Action ตรงสำหรับ **AI Agent Platform builders**:
  - (1) **SI-firm channel readiness** — เปิด partner certification program ภายใน Q3 + onboarding kit สำหรับ consultant (deploy ในชั่วโมง ไม่ใช่สัปดาห์) — ก่อน PwC/Globant/EY ลงตัวกับ vendor ใหญ่ทั้งหมด
  - (2) **MCP-native integration** — implement MCP server + client เป็น first-class สำหรับ ServiceNow, Salesforce, SAP, Workday, Jira ภายในไตรมาส — vendor ที่ require custom SDK จะหลุดจาก enterprise procurement
  - (3) **AgentOps observability + experiment context** — feature "agent eval + iteration recommendation" แบบ ARIA — สำหรับ team ที่ scale agent หลายสิบตัวใน prod. Data layer เป็น differentiation
  - (4) **Vertical agent for regulated industry** — pre-built workflow + rule engine + audit + human-in-the-loop primitive สำหรับ banking/insurance/healthcare — เป็น category ที่ funding ไหลแรงสุด
  - (5) **Neutral orchestrator positioning** — ขยายเป็น control plane ที่ orchestrate agent จาก vendor หลายราย + open governance — ลูกค้า enterprise ระยะกลางจะปฏิเสธ "single-vendor agent overlord"

- Action ตรงสำหรับ **Business / users** ที่ deploy AI Agent:
  - (1) **เช็ค channel partner ก่อน vendor** — ดูว่า SI ใน region (Globant, PwC, EY, Accenture, KPMG, local SI ในไทย) มี certified team สำหรับ Claude/GPT/Gemini ตามที่ vendor ที่จะ deploy หรือไม่ — ถ้าไม่มี time-to-production จะยาวกว่าที่ vendor บอก 3-6 เดือน
  - (2) **Insist on MCP-native** ใน RFP — vendor ที่ยัง require custom SDK สำหรับ enterprise tool ใหญ่ = red flag governance + integration cost จะ pop ภายหลัง
  - (3) **Pilot vertical agent (regulated) ผ่าน buy ก่อน build** — Taktile + คู่แข่งใน financial services ลดเวลา 6-12 เดือน + มี certified architecture. แต่ engage regulator ในไทย (BoT, SEC, OIC) ตั้งแต่แรก
  - (4) **AgentOps layer ตั้งแต่ pilot** — W&B Weave / ARIA / Langfuse / Braintrust — observability + experiment iteration คือ piece ที่ทำ pilot → production ต่างกัน 5-10x ROI
