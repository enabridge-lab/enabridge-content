รอบนี้มี 4 เรื่อง เน้น Agentic AI, business use case, และ trend ที่เอามาใช้กับ OpenBridge ได้

เรื่องที่ 1
วันที่ 1 พฤษภาคม Pentagon ประกาศเซ็นดีล AI พร้อมกัน 7 บริษัท ให้รันบน classified network ของ DoD ได้แก่ SpaceX, OpenAI, Google, NVIDIA, Reflection, Microsoft, และ AWS แต่ที่หายไปคือ Anthropic — สัญญา 200 ล้านดอลลาร์ที่ได้ตอนกรกฎาคมปีที่แล้วถูกยกเลิกอย่างเป็นทางการ เหตุผลคือ Anthropic ยึดเงื่อนไขห้ามใช้ Claude สำหรับ mass surveillance หรือ autonomous weapons targeting แต่ Pentagon ยึด "all lawful uses" ส่วน OpenAI กับ Google ยอมเงื่อนไขนี้ไป ที่ผมคิดว่าน่าสนใจคือ paradox ที่ใหญ่มาก — Anthropic เพิ่งจะปิด round ที่ valuation 900 พันล้านดอลลาร์ ใหญ่กว่า OpenAI แล้ว แต่ขายให้รัฐบาลตัวเองไม่ได้ Reflection AI ที่ติดลิสต์ตัว classified deal ก่อตั้งแค่ปี 2024 ยังไม่มี product commercial แต่ได้ classified contract แล้ว นี่คือ signal ว่า lab ที่ pick a side ชัดได้สิทธิ์เหนือ lab ที่ใหญ่ที่สุด สำหรับ OpenBridge สัปดาห์ที่แล้วเราพูดเรื่อง vendor neutrality + policy disclosure layer ตอนนี้ signal นี้ confirmed หนักมาก ลูกค้า bank, insurer, healthcare ที่ทำงาน defense supplier จะมี RFP ใน 60-90 วันถามว่า vendor ของคุณมี US designation status อะไร switching path เร็วแค่ไหน OpenBridge มี window 6 เดือนสร้าง AI vendor compliance dashboard ขาย bank, insurer ในไทยเป็น SaaS subscription ราคา 30 ถึง 100 พันดอลลาร์ต่อปีต่อ tenant — recurring revenue ที่ติด stickiness ระดับ regulatory ครับ

เรื่องที่ 2
ปลายเมษา Anthropic เปิดดีลใหญ่ที่สุดในประวัติศาสตร์ private market — รับข้อเสนอจากนักลงทุนหลายราย raise 50 พันล้านดอลลาร์ที่ valuation 850 ถึง 900 พันล้าน ตั้ง deadline 48 ชั่วโมงให้ submit allocation คาดว่าปิดดีลใน 2 สัปดาห์ ถ้าปิดที่ 900 พันล้าน Anthropic แซง OpenAI ที่ 852 พันล้านจากเดือนกุมภา ขึ้นเป็น AI lab อันดับ 1 โลก ตัวเลขที่ Anthropic เปิดเอง — run-rate revenue ทะลุ 30 พันล้านดอลลาร์ ขึ้นจาก 9 พันล้านตอนสิ้นปีที่แล้ว เติบโต 3.3 เท่าใน 4 เดือน Claude เข้า base lineup ของ IBM Bob 80,000 คน, Salesforce Agentforce, Cloudflare agent-as-customer พร้อมกัน paradox ที่ผมว่าน่าสนใจสุดคือ ห่างกันแค่ 24 ชั่วโมง 1 พฤษภา Anthropic ถูก Pentagon ตัดออกจาก classified network deal สัญญา 200 ล้านโดน void แต่ตลาด private valuation ไม่ได้ price in เลย เพราะ public sector revenue ของ Anthropic คาด <2% ของ ARR — federal exclusion = noise ใน $30B run-rate สำหรับ OpenBridge สัญญาณนี้ชัดมาก สมรภูมิ model layer จบแล้วในระดับ winner-take-most — Anthropic, OpenAI, Google มี run-rate รวมเกิน 70 พันล้าน ไม่มีที่ให้ Thai LLM หรือ regional foundation model lab เกิดอีกแล้ว OpenBridge ต้อง position เป็น orchestration + integration layer ไม่ใช่ model layer และเร่ง enterprise sales motion เพราะ Anthropic ARR explosion = leading indicator ของ enterprise budget ที่จะ flow down ถึง Thai subsidiary ของ MNC ใน Q2 ถึง Q3 ครับ

เรื่องที่ 3
วันที่ 29 เมษา Salesforce ปล่อย Agentforce Operations ขึ้น GA ทันที เป็น extension ของ Agentforce ที่ขยายจาก front-office ลงสู่ back-office — finance, operations, compliance, HR ฟีเจอร์ที่ผมว่าน่าสนใจสุดคือ agent "complete the work" ไม่ใช่แค่ orchestrate — สามารถ extract data จาก document, run computation, update ERP, flag compliance gap ขึ้น human reviewer เฉพาะตอน need approval อีกอันคือ process digitization ผ่าน upload — user upload Lucidchart, Word doc, หรือแม้กระทั่ง drawing ของ workflow แล้วระบบแปลงเป็น multi-step agent workflow + มี 30 pre-built blueprints สำเร็จ ตัวเลขที่ Salesforce อ้าง — ลด cycle time 50 ถึง 70%, ลด manual task 80% สเกล Agentforce ตอนนี้ — 18,500 ลูกค้า, 9,500 paid deal, ARR รวม Data 360 แตะ 1.4 พันล้านดอลลาร์ +114% YoY = product line โตเร็วที่สุดในประวัติ Salesforce นี่คือ playbook คลาสสิก — front-office ก่อน, ได้ data + reference, ขยายเข้า back-office ที่ TAM ใหญ่กว่า 3-4 เท่า ในอีก 24 เดือนน่าจะแตะ 5 ถึง 7 พันล้าน ARR สำหรับ OpenBridge — Salesforce reseller ASEAN (Aware, AIS Cloud) จะมี Agentforce Operations ใน 3 ถึง 6 เดือน mid-market Thai ที่ยังไม่ได้ใช้ Salesforce ราว 70% คือ window ของ OpenBridge ต้อง ship MVP ของ back-office agent ที่ใช้ document เป็น input ภายใน 90 วัน + productize 5 vertical blueprints — AP/AR reconciliation, customs declaration, invoice 3-way match, employee onboarding, compliance log สำหรับ BoT submission ราคา 200 ถึง 1,500 ดอลลาร์ต่อเดือนต่อ blueprint และเปลี่ยน sales pitch จาก integration เป็น process digitization — ลูกค้าไทย mid-market พูดภาษา "ลด manual work" ไม่ใช่ "iPaaS connector" ครับ

เรื่องที่ 4
Sierra ของ Bret Taylor — อดีต CTO Facebook, co-CEO Salesforce, chair OpenAI board — เคลื่อน 2 step ใหญ่ในเดือน เม.ย. step แรก 10 เม.ย. ปล่อย Ghostwriter ตัวที่เป็น meta-agent — agent ที่ build agent อื่นจาก natural language description แทนที่จะเขียน prompt หรือ flow เอง user แค่บอก "ฉันต้องการ agent จัด refund ลูกค้าติดต่อ 2 ครั้งใน 7 วัน" → Ghostwriter generate spec, deploy agent, ตั้ง integration, เริ่ม shadow mode สัปดาห์แรก step สอง 23 เม.ย. Sierra ซื้อ Fragment สตาร์ทอัพฝรั่งเศสจาก YC เป็นการ acquire ครั้งที่ 3 ในปีนี้ ก่อนหน้านี้ซื้อ Opera Tech ที่ญี่ปุ่น + Receptive AI ที่ทำ voice ตัวเลข Sierra ตอนนี้ ARR 150 ล้านดอลลาร์ +50% ใน 5 เดือน, valuation 10 พันล้าน, voice surpass text เป็น primary channel processed Bret Taylor ขึ้นเวที Stratechery บอก "era of clicking buttons is over" software ในยุคหน้าไม่มี UI ปกติ มีแต่ agent ที่คุยภาษาคนแล้วทำ task ให้ สำหรับ OpenBridge — path ที่ห้ามเดิน คือ build generic agent builder แข่ง Sierra/Salesforce/Microsoft จะแพ้แน่ path ที่ควรเดิน คือ vertical-specific agent factory เช่น Thai AP/AR factory ที่เข้าใจ VAT 7% + withholding tax, customs declaration factory ที่ filing ผ่าน NSW อัตโนมัติ, BoT/SEC compliance agent factory และอย่าลืม voice — ทุก vendor ระดับโลก voice surpass text ใน Q1 ถ้า OpenBridge ยังเน้น text อย่างเดียวจะ miss 60-80% ของ enterprise volume ในไทย partner กับ Vapi, ElevenLabs Thai, Looloo สำหรับ voice infra ภายใน 60 วันครับ
